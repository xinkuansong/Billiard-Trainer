## 四、用户流程

### 流程 1：完整角度测试（20 题）

**步骤**：
1. 角度 Tab → 点击「开始测试」
2. 第 1 题：观察球台 Canvas（目标球 + 母球 + 袋口），估算角度
3. 数字键盘输入角度值（如 30）→ 点击「确认」
4. 查看答案动画：绿色正确路线 + 橙色用户路线 + 红点 + 误差值
5. 点击「下一题」
6. 重复步骤 2-5，连续做完 20 题
7. 测试完成后查看结果摘要

**预期结果**：20 题流畅完成，角度计算正确

- [ ] 每题球台布局随机变化（母球位置、目标球位置不同）
- [ ] 角袋/中袋交替出现（约 60%/40%）
- [ ] 误差 ≤ 3° 显示绿色「精准」
- [ ] 误差 3°–10° 显示橙色「接近」
- [ ] 误差 > 10° 显示红色「偏差较大」
- [ ] 20 题全部完成后有结果摘要

### 流程 2：角度计算手动验证

**步骤**：
1. 找到一道 45° 角的题目（可多做几题等遇到接近 45° 的）
2. 输入 45 提交
3. 查看正确答案是否接近 45°
4. 验证偏移百分比是否接近 70.7%（sin45° ≈ 0.707）

**预期结果**：角度和偏移值计算正确

- [ ] 45° 偏移 ≈ 70.7%R
- [ ] 30° 偏移 ≈ 50%R（如遇到 30° 题）

### 流程 3：进球点对照表

**步骤**：
1. 角度 Tab → 点击「对照表」
2. 拖动滑块从 0° 到 90°，观察球面接触点实时移动
3. 将滑块拖到 45°，确认偏移百分比约 70.7%
4. 查看静态对照表 13 行（0°/10°/15°…90°）
5. 确认每行含通称标记（全球/二分之一球/四分之三点等）

**预期结果**：对照表数据准确，交互流畅

- [ ] 滑块拖动时 Canvas 实时更新
- [ ] 45° 偏移显示约 70.7%
- [ ] 13 行标准角度数据完整
- [ ] 通称标记可见（全球/半球等）
- [ ] 原理说明可见

### 流程 4：历史记录查看

**步骤**：
1. 先完成流程 1（20 题测试）
2. 角度 Tab → 「测试历史」入口
3. 查看 2×2 统计卡（总测试数/平均误差/精准率/最近趋势）
4. BTSegmentedTab 切换时间范围（周/月/全部）
5. 查看误差趋势折线图
6. 切换角袋/中袋趋势
7. 查看角度区间分析柱状图

**预期结果**：历史数据正确记录

- [ ] 2×2 统计卡数值与测试记录一致
- [ ] 时间范围切换后趋势与统计合理更新
- [ ] 误差趋势折线图有数据展示（或数据不足时显示「数据不足」）
- [ ] 角袋/中袋 toggle 切换后趋势对应变化
- [ ] 角度区间分析柱状图展示各范围正确率
- [ ] 数据在 App 重启后保持

### 流程 5：自适应出题验证

**步骤**：
1. 连续做 20 题测试
2. 故意在某个角度范围（如 60°-70°）输入大幅错误答案（如差 30°+）
3. 在下一轮 20 题中观察该角度范围出题频率是否增加

**预期结果**：连续答错某区间后该区间出题频率明显增加

- [ ] 答错较多的角度区间在后续测试中出现更频繁

### 流程 6：每日限制（免费用户）

**步骤**：
1. 确保未购买 Pro
2. 连续做测试，第 20 题提交后观察
3. 继续第 21 题，确认是否出现限制提示

**预期结果**：第 21 题出现 BTPremiumLock 引导订阅

- [ ] 前 20 题正常答题
- [ ] 第 21 题出现每日限制提示
- [ ] 提示包含「升级 Pro」引导按钮
