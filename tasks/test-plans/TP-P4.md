## 四、用户流程

### 流程 1：按计划训练（完整流程）

**步骤**：
1. 启动 App → 训练 Tab
2. 训练首页显示空状态 → 点击引导按钮进入计划列表
3. 选择「beginner（入门基础）」计划 → 进入计划详情
4. 点击「开始此计划」激活计划
5. 返回训练首页 → 确认「今日训练」显示 Drill 列表
6. 点击「开始今日训练」
7. 在第一个 Drill 记录界面输入：目标球数 10，进球数 7
8. 点击「+ 添加一组」→ 输入：目标球数 10，进球数 8
9. 点击「完成此 Drill」→ 滑到第二个 Drill
10. 录入 1 组数据 → 完成
11. 心得页面输入「练习手感不错」→ 点击保存
12. 总结页查看统计数据

**预期结果**：完整训练流程顺畅，数据正确

- [ ] 从首页到第一个 Drill 录入 <= 3 步操作
- [ ] 每组成功率实时计算正确（如 7/10 = 70%）
- [ ] 总结页整体成功率与各组数据一致
- [ ] 「保存训练」按钮可点击

### 流程 2：自由记录模式

**步骤**：
1. 训练 Tab → 点击「自由记录」
2. 进入空白训练界面 → 手动添加一个 Drill
3. 录入 1 组数据 → 完成
4. 跳过心得 → 查看总结 → 保存

**预期结果**：无计划情况下也能正常完成训练记录

- [ ] 自由记录入口可正常进入
- [ ] 可手动添加 Drill
- [ ] 跳过心得后训练仍能保存

### 流程 3：付费计划锁定

**步骤**：
1. 计划列表 → 点击一个付费计划（如 intermediate）
2. 确认详情页有 BTPremiumLock 遮罩
3. 点击「解锁全部内容」

**预期结果**：付费计划不可直接使用

- [ ] 付费计划有遮罩
- [ ] 免费计划（beginner/cueball）无遮罩，可正常激活

### 流程 4：训练持久化验证

**步骤**：
1. 完成流程 1 并保存
2. 强制退出 App（上滑关闭）
3. 重新打开 App
4. 检查训练首页今日已完成状态

**预期结果**：训练数据持久化成功

- [ ] 重启后今日训练项有完成标记
- [ ] 数据未丢失
