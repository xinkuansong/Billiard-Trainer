## 四、用户流程

### 流程 1：日历浏览与训练详情

**步骤**：
1. 历史 Tab → 默认显示当月日历
2. 确认有训练记录的日期显示分类胶囊标签
3. 点击有记录的日期
4. 查看当日训练列表（日期下方）
5. 点击某条训练记录
6. TrainingDetailView Sheet 弹出
7. 查看统计横滚（进球/组/分钟/时段/日期）
8. 查看 Drill 组明细（缩略图 + 名称 + 每组成绩）
9. 如有心得备注，确认展示

**预期结果**：日历→列表→详情流程顺畅

- [ ] 有记录的日期正确标记
- [ ] 点击日期后训练列表展示
- [ ] 详情 Sheet 完整展示所有信息
- [ ] 各 Drill 每组「进X/目标X」数据正确
- [ ] 心得内容（如有）正确展示

### 流程 2：月份翻页

**步骤**：
1. 当月日历 → 点击上月按钮
2. 确认切换到上月日历
3. 继续翻到更早月份（如无数据的月份）
4. 点击下月按钮返回

**预期结果**：翻月正常，无数据月份不崩溃

- [ ] 翻月日历正确切换
- [ ] 无数据月份日历正常展示（无标记）
- [ ] 返回当月按钮正常

### 流程 3：统计图表查看

**步骤**：
1. BTSegmentedTab 切换到「统计」
2. BTTogglePillGroup 默认显示「周」
3. 查看三张统计卡片（训练概况/时长/分类成功率）
4. 查看训练频率柱状图 + 趋势线
5. 切换到「月」→ 观察图表更新
6. 切换到「年」→ 观察图表更新
7. 查看 2 列网格分类成功率（8 大类）

**预期结果**：统计数据正确，图表随时间范围切换

- [ ] 三张统计卡片有数据（非全空）
- [ ] 柱状图有数据条（琥珀色时长 + 绿色成功率）
- [ ] 趋势线叠加正确
- [ ] 周/月/年切换后图表刷新
- [ ] 2 列网格 8 大类显示（含环比变化 + 迷你柱状图）

### 流程 4：空数据统计

**步骤**：
1. 全新安装（无训练记录）
2. 历史 Tab → 切换到「统计」

**预期结果**：BTEmptyState 展示引导

- [ ] 空状态视图「开始第一次训练」显示
- [ ] 不显示空白图表

### 流程 5：Freemium 60 天限制

**步骤**：
1. 确保有一条 > 60 天前的训练记录（可通过修改系统日期或使用预置测试数据）
2. 免费用户身份查看该记录
3. 确认锁定提示出现

**预期结果**：60 天前的记录被锁定

- [ ] 60 天内记录正常访问
- [ ] 60 天前记录显示锁图标 + Pro 标签
- [ ] 点击锁定记录触发 SubscriptionView

### 流程 6：无记录日期点击

**步骤**：
1. 日历上点击一个没有训练记录的日期

**预期结果**：不崩溃，列表区域为空

- [ ] 不崩溃
- [ ] 列表区域无内容（或提示「当天无训练记录」）
