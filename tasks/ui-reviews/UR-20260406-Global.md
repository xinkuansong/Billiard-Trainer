### U-01 History Tab NavigationStack 缺失 navigationDestination
- **类别**：布局 / 产品规格
- **严重程度**：P1（功能缺陷）
- **位置**：MainTabView > Tab 4（历史）
- **现状**：`MainTabView.swift` 第 45-52 行 History Tab 的 NavigationStack 没有注册 `.navigationDestination(for: HistoryRoute.self)` 修饰符。`historyDestination(for:)` 函数（103-110 行）已定义但为死代码，从未被调用。
- **预期**：History Tab 的 NavigationStack 应注册 HistoryRoute 导航目标，使得 TrainingDetailView、StatisticsView 等子页面可通过 `router.historyPath` 正确 push。
- **修复方向**：在 NavigationStack 内添加 `.navigationDestination(for: HistoryRoute.self) { route in historyDestination(for: route) }`
- **路由至**：swiftui-developer
- **代码提示**：`QiuJi/App/MainTabView.swift` L45-52, L103-110

### U-02 Profile Tab 缺少 NavigationStack 包装
- **类别**：布局 / 产品规格
- **严重程度**：P1（功能缺陷）
- **位置**：MainTabView > Tab 5（我的）
- **现状**：`MainTabView.swift` 第 54-58 行 ProfileView 直接作为 Tab 内容，无 NavigationStack 包装。`AppRouter` 声明了 `profilePath` 属性但未被使用。若 ProfileView 内部有基于 `router.profilePath` 的 push 导航（如 FavoriteDrillsView、SettingsView），将无法工作。
- **预期**：与其他 4 个 Tab 一致，使用 `NavigationStack(path: $router.profilePath)` 包装。
- **修复方向**：为 ProfileView 添加 NavigationStack 包装并注册相应 navigationDestination。如果 ProfileView 内部自行管理 NavigationStack 则需确认 router.profilePath 可移除。
- **路由至**：swiftui-developer
- **代码提示**：`QiuJi/App/MainTabView.swift` L54-58; `QiuJi/App/AppRouter.swift` L38

### U-03 BTTogglePillGroup Dark Mode 选中态不符合 §6.3 规范
- **类别**：Dark Mode
- **严重程度**：P1（功能缺陷）
- **位置**：Core/Components > BTTogglePillGroup
- **现状**：BTTogglePillGroup 选中态在所有模式下均使用 `btPrimary` 填充 + 白字。Dark Mode 下表现为绿底白字。
- **预期**：UI-IMPLEMENTATION-SPEC §6.3 规定 Dark Mode 筛选 Chip 选中态为 `#F2F2F7` 填充 + `#000000` 文字；未选中 `#2C2C2E` 填充 + `rgba(235,235,240,0.6)` 文字 + `#3A3A3C` 描边。已知偏差 D-1 明确修正此项。
- **修复方向**：在 BTTogglePillGroup 中检测 `colorScheme`，Dark Mode 下选中态改为 `Color(UIColor.systemGray6)` (≈ `#F2F2F7`) 填充 + `.btText`（黑字）；未选中态改为 `btBGTertiary` 填充 + `btTextSecondary` + `btBGQuaternary` 描边。
- **路由至**：swiftui-developer
- **代码提示**：`QiuJi/Core/Components/BTTogglePillGroup.swift` L17-27

### 审查总结
- 审查文件数量：约 20 个（2 App 级 + 15 组件 + 3 DesignSystem）
- 发现问题：11 项（P0: 0 / P1: 3 / P2: 8）
- 14 项已知偏差：12 项已修正 ✅，1 项未修正 ❌（D-1），1 项部分修正 ⚠️（L-4）

- 建议下一步：
  1. **优先修复 P1**：U-01 History 导航、U-02 Profile 导航、U-03 Dark Chip 选中态
  2. **顺手修复 P2**：U-04/U-05 字体微调、U-10 文档同步
