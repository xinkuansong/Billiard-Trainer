# 功能规划

## 1. 功能全景（模块总览）

| **功能模块** | **核心功能** | **优先级** |
|---|---|---|
| **动作库** | 台球Drill内容库（动画图解+视频）、分类浏览、搜索、收藏 | P0 |
| **训练记录** | 快速记录一次练球、每个Drill的组数/进球数（数字键盘输入）、心得备注 | P0 |
| **训练计划** | 官方预设计划（分级别）、用户自定义计划 | P0 |
| **角度感知训练** | 角度测试（随机出题）、进球点对照表（交互式滑块+完整表格） | P0 |
| **历史与统计** | 日历视图、训练详情、成功率趋势、训练频率图表 | P1 |
| **账号系统** | 微信 / 短信 / Sign in with Apple、个人信息、数据云同步（自建 REST API） | P0 |
| **个人设置** | 球种偏好、训练目标、通知提醒 | P1 |
| **AI摄像分析** | 摄像头识别击球轨迹、实时反馈 | v2.0 |
| **培训机构B端** | 学员管理、课程分配 | v2.0 |

---

## 2. 用户故事与验收标准

### F1 — 动作库：浏览与查看 Drill

- **故事**：作为台球爱好者，我想在动作库里按分类浏览所有训练项目，以便找到我想学习或练习的具体技术
- **验收**：
  - 可按8个大类 + 子分类浏览全部Drill
  - 可通过顶部球种筛选（中式台球/9球/通用）过滤
  - 可搜索Drill名称
  - 每个Drill详情页展示：球台示意图、教学视频（或占位）、文字说明、难度标签、训练建议
- **优先级**：P0
- **依赖**：Drill内容数据，初期约60–70项

---

### F2 — 动作库：Drill 收藏

- **故事**：作为爱好者，我想收藏常用的训练项目，以便下次快速找到而不用重新翻分类
- **验收**：
  - Drill 详情页有「收藏」按钮（心形图标）
  - 动作库底部或「我的」Tab 下可访问收藏夹
- **优先级**：P1
- **依赖**：F1

---

### F3 — 训练计划：选择与使用官方计划

- **故事**：作为初学者，我想选择一个系统的官方训练计划，以便每次去球馆前知道该练什么
- **验收**：
  - 可浏览官方计划列表（按入门/初级/中级/高级分组）
  - 计划详情页展示：总周期、训练目标、每天的Drill组合
  - 可「开始此计划」，之后在训练Tab显示今日安排
  - 按天展示已完成/待完成状态
- **优先级**：P0
- **依赖**：F1、计划内容数据

---

### F4 — 训练计划：自定义计划

- **故事**：作为进阶玩家，我想自己选择Drill组合成个人计划，以便按自己的训练体系练习
- **验收**：
  - 可新建自定义计划，命名 + 设置每周训练天数
  - 每天可从动作库添加多个Drill，设置每个Drill的目标组数和每组球数
  - 可保存并激活自定义计划
- **优先级**：P1
- **依赖**：F1、F3

---

### F5 — 训练记录：开始并完成一次训练

- **故事**：作为爱好者，我想在球馆练球时随手记录每个Drill的完成情况，以便事后复盘和追踪进步
- **验收**：
  - 支持「按计划」和「自由记录」两种模式
  - 每个 Drill 可记录：组数、每组完成球数（如「15球进了11球」）、自评（可选）
  - 可计时或手动输入练习时长
  - 可添加心得备注（纯文字，可跳过）
  - 完成后显示本次训练总结（时长/项目数/平均成功率）
  - **操作目标**：3步内进入第一个Drill记录界面
- **优先级**：P0
- **依赖**：F3（计划），或独立使用

---

### F6 — 历史：日历视图与训练详情

- **故事**：作为爱好者，我想在日历上看到自己的训练记录，以便直观感受到训练频率和坚持情况
- **验收**：
  - 历史 Tab 默认显示当月日历，有训练记录的日期高亮标记
  - 点击日期显示当天训练详情（Drill列表 + 各组成绩 + 心得）
  - 支持按月翻页浏览
- **优先级**：P0
- **依赖**：F5

---

### F7 — 统计：训练数据分析

- **故事**：作为进阶玩家，我想查看自己在各类Drill的成功率趋势，以便发现薄弱环节并调整训练重心
- **验收**：
  - 统计视图支持周/月/年时间范围切换
  - 展示：训练天数、总时长、总组数
  - 训练频率折线图
  - 各大类训练时长/次数分布（柱状图或雷达图）
  - 单个Drill的历史成功率趋势可查（从Drill详情页进入）
- **优先级**：P1
- **依赖**：F6

---

### F8 — 账号：Sign in with Apple + 数据同步

- **故事**：作为用户，我想用 Apple 账号登录并让训练数据自动云同步，以便换设备后数据不丢失
- **验收**：
  - 首次打开 App 提示登录（可跳过，离线使用）
  - 支持 Sign in with Apple
  - 登录后训练记录、收藏、计划自动通过自建 REST API 同步
  - 多设备登录同一账号数据一致
- **优先级**：P0
- **依赖**：无

---

### F9 — 个人设置：球种偏好与训练目标

- **故事**：作为用户，我想设置我常打的球种和每周训练目标，以便 App 为我提供更相关的推荐
- **验收**：
  - 可设置主打球种（中式台球/9球/两者都打）
  - 可设置每周训练天数目标（如「每周3次」）
  - 首次使用时通过引导流程完成设置
- **优先级**：P1
- **依赖**：F8

---

### F10 — 角度感知训练：角度测试

- **故事**：作为台球爱好者，我想通过反复判断随机球形的切入角度，训练自己的角度视觉感知能力，以便在实战中能更准确地目测角度并找到对应的进球点
- **验收**：
  - 生成随机题目：顶视图球台，目标球带袋口方向标记，母球位置随机（切入角在5°–85°内，以5°为精度）
  - 用户通过数字键盘输入估计角度，点击「确认」
  - 结果动画：高亮正确击球路线 + 在目标球面上标注正确接触点位置
  - 显示误差（用户答案 vs 正确答案）
  - 可连续练习，支持设置难度（容差范围±5°/±10°）
  - 历史测试结果可在「角度」Tab 内查看（平均误差趋势）
- **优先级**：P0（角度 Tab 核心功能）
- **技术说明**：纯本地计算（sin函数），无需网络，无需后端支持

---

### F11 — 角度感知训练：进球点对照表

- **故事**：作为台球爱好者，我想查阅不同切入角对应的目标球接触点位置，以便理解角度瞄准法的原理并用于实战
- **验收**：
  - 交互式滑块（0°–90°）：拖动时目标球示意图实时更新接触点位置（动画）
  - 数值显示：当前角度 + 接触点偏移（占半径的百分比 = sin(α)×100%）
  - 完整对照表：列出主要角度（0°/10°/15°/20°/25°/30°/35°/40°/45°/48.6°/60°/75°/90°）及对应接触点
  - 每行含小型球面示意图（标注接触点位置）
  - 原理说明区：简要解释角度瞄准法数学原理（偏移 = sin(α) × R）
- **优先级**：P0（角度 Tab 核心功能）
- **技术说明**：全部预计算，静态数据，零网络依赖

## 3. MVP 范围（v1.0）

**包含功能**：F1（动作库浏览）、F3（官方计划）、F5（训练记录）、F6（历史日历）、F8（账号同步）、**F10（角度测试）**、**F11（进球点对照表）**

**尽量包含（时间允许）**：F2（收藏）、F4（自定义计划）、F7（统计）、F9（偏好设置）

**内容量目标**：
- 动作库：约60–72个Drill（覆盖8大类，含图解；视频内容逐步填充）
- 官方计划：至少4套（入门/初级/中级/综合提升）

## 5. 范围外 / 暂缓

| 功能 | 暂缓原因 |
|---|---|
| 在线对战/游戏功能 | 与产品定位冲突，非训练工具 |
| 社交广场/关注/动态 | 增加复杂度，分散训练专注度 |
| AI视频内容自动生成 | 运动技术精度要求高，质量难保障 |
| 斯诺克内容 | v1.0先聚焦中式台球+9球，后续扩展 |
| 跳球教学 | 在大多数业余场所受限，暂不纳入 |
