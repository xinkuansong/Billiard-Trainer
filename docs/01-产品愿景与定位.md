# 产品愿景与定位

## 1. 背景与问题

### 1.1 市场背景

中国国内台球热度持续上升，以斯诺克、中式台球为代表的球类运动在业余爱好者群体中普及率快速提高。

### 1.2 现有替代品的核心缺口

| 现有替代品 | 满足的需求 | 核心缺陷 |
|---|---|---|
| 专业培训机构 | 系统课程 + 实体场地 + 专业指导 | 定位精英/幼训运动员，门槛高，不面向大众爱好者 |
| 短视频平台（抖音/B站等） | 海量免费教学内容 | 无法系统记录进度；内容碎片、不成体系；注意力干扰严重，难以专注跟练 |
| 台球类游戏 App | 娱乐对局、模拟练球 | 训练属性弱，缺乏真实场景数据记录与训练规划 |

**核心痛点**：大量台球爱好者有提升意愿，但缺乏一个能把「系统知识 + 训练规划 + 进度记录」整合在一起、专注于训练场景的工具。

## 2. 产品愿景

打造一款面向台球爱好者与进阶玩家的 **训练伴侣 App**，整合系统化知识教学、个性化训练计划、训练数据记录与复盘，帮助用户在真实练球场景中持续提升技术水平。

中长期愿景：成为中文台球爱好者群体认可的首选训练工具，并向培训机构提供学员管理能力。

## 3. 目标用户与定位

### 初步三类用户（优先级待确认）

| 用户类型 | 核心需求 | 使用场景 | 特点 |
|---|---|---|---|
| **普通爱好者** | 轻量入门教学 + 简单训练记录 | 业余打球时随手记录 | 基数最大，付费意愿待验证 |
| **进阶/职业玩家** | 精细化训练数据记录 + 训练计划管理 | 高频系统训练 | 需求明确，愿意为专业工具付费 |
| **培训机构** | 学员管理 + 课程分配 + 教学内容 | B 端管理场景 | 客单价高，但产品逻辑为 B 端，开发复杂度大 |

> ⚠️ **待确认**：MVP 主力用户为哪类？三类同时服务可能导致初版过重。

### 与现有替代品的区隔

- vs 短视频平台：专注训练场景，减少干扰；知识结构化；可记录进度
- vs 专业培训机构：线上化、低门槛、自主节奏
- vs 游戏 App：真实训练数据驱动，非娱乐模拟

## 4. 价值主张

**草案（待精炼）**：「在台球馆练球时，随时记录、系统跟进、持续进步的训练伙伴。」

支撑论点：
- 打球时即时记录，不依赖记忆
- 系统化知识与个性化训练计划结合
- 减少短视频干扰，专注训练本身
