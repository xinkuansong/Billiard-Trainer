# 竞品分析

## 1. 竞品范围

- **直接竞品**（同为台球训练/工具类）：国内暂无；国际：DrillRoom、WPB、ChalkySticks Trainer
- **间接竞品**（台球娱乐/游戏）：台球世界、天天台球、台球大师3D（国内主流）
- **参照竞品**（运动训练工具通用模式）：keep、运动类健身App

## 3. 可借鉴点与差异化

### 从 DrillRoom 可借鉴
- Drill 结构化设计思路（按技术点分类：击球准度、母球控制、走位等）
- 训练数据可视化方式（热力图、进球率趋势）
- 订阅制商业模式在训练工具类App的可行性验证

### 本产品差异化路径
1. **中文化 + 本土内容**：面向斯诺克/中式台球爱好者（DrillRoom几乎不含这两个球种内容）
2. **系统知识库**：理论知识（站位/握杆/发力/母球控制原理），DrillRoom只有Drill没有理论解释
3. **训练日志 + 心得记录**：个人训练过程的完整记录，所有竞品均未重点做此功能
4. **AI摄像分析**（v2.0）：用国内球种（中式台球台面尺寸、斯诺克球台）训练的AI模型

## 4. 风险与机会

| | 说明 |
|---|---|
| **机会** | 国内市场完全空白，无直接竞品；中文+斯诺克/中式台球是DrillRoom的盲区 |
| **风险1** | DrillRoom若推出中文版，直接降维打击核心功能 |
| **风险2** | 国内短视频平台（抖音/B站）若推出站内工具功能，会削弱知识内容差异化 |
| **风险3** | 教学内容质量：AI生成内容若不准确，损害用户信任 |
