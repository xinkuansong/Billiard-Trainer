# 信息架构与交互设计

## 1. 产品设计参照

参照「训记」App（健身训练记录）的产品模式，将台球训练中的每个 Drill（训练项目）视同健身「动作」，建立完整的「台球动作库」。

**训记 → 台球 App 概念映射：**

| 训记概念 | 台球 App 对应 |
|---|---|
| 健身动作（如「杠铃卧推」） | 台球训练项目（如「五分点定杆练习」） |
| 动作分类（胸/背/腿） | 训练分类（准度/杆法/走位） |
| 训练计划（胸肌日） | 台球训练计划（今日专项走位训练） |
| 组数×重复次数 | 组数×每组球数（如「3组×15球」） |
| 进度（力量增长） | 进度（完成率/成功率提升） |

## 2. 主导航结构（5 Tab）

```
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│  训练   │  动作库  │  角度   │  历史   │  我的   │
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

| Tab | 核心功能 | 使用频率 |
|---|---|---|
| 训练 | 按计划练球 + 快速记录 | 每次练球必用 |
| 动作库 | 浏览/查看 Drill 内容 | 中频（查资料、选Drill） |
| **角度** | **角度感知训练 + 进球点对照表** | **中频（专项感知训练）** |
| 历史 | 日历回顾 + 数据统计 | 低频（复盘时） |
| 我的 | 个人信息、设置、账号 | 低频 |

## 3. 各 Tab 信息架构

### Tab 1 — 训练（Training）

**定位**：日常最高频入口，引导用户完成每次练球，快速记录。

```
训练
├── 今日训练卡
│   ├── 当前计划名称（如「初级走位专项 - 第3天」）
│   ├── 今日计划内容预览（含Drill列表）
│   └── [开始训练] 按钮 → 进入训练记录流程
│
├── 训练计划
│   ├── 官方计划（按难度/目标分类）
│   │   ├── 入门阶段：「新手入门8周计划」
│   │   ├── 初级阶段：「基础杆法专项」「走位入门计划」
│   │   ├── 中级阶段：「加塞进阶计划」「分离角专项」
│   │   └── 综合提升：「全能训练计划」
│   └── 我的计划（用户自定义，从动作库选Drill组合）
│
└── 快速记录（不按计划，自由本次练了什么）
    ├── 选择球种（中式台球 / 9球）
    ├── 添加训练项目（从动作库选择）
    │   └── 每项：组数、每组球数、完成数、自评
    ├── 练习时长（自动计时或手动填写）
    ├── 心得备注（自由文本）
    └── 保存
```

**训练记录流程（核心交互）：**

```
开始训练
  → 当前训练项目卡片（全屏/半屏）
    → 显示 Drill 名称 + 球台动画（母球/目标球路线动画）+ 要点提示
    → 记录本组完成数：数字键盘输入「进了X球 / 共Y球」
    → [完成本组] → [下一组] 或 [下一个项目]
      → 完成所有项目
        → 训练完成页（总结 + 写心得入口）
```

### Tab 3 — 角度（Angle Training）⭐ 产品独创功能

**定位**：训练玩家对击球角度的视觉感知能力，配合「进球点对照表」提供角度瞄准法的完整理论工具。

**核心数学原理**：对于切入角 α，从母球方向看目标球，接触点偏离球心的距离 = R × sin(α)（R为球半径）。

```
角度
├── 角度测试（Angle Quiz）
│   ├── 顶视图球台
│   │   ├── 目标球（含袋口方向箭头）
│   │   ├── 母球（位置随机生成，切入角随机）
│   │   └── 题目：「这个切入角是多少度？」
│   ├── 用户输入方式
│   │   ├── 数字键盘输入角度估值
│   │   └── [确认] 按钮
│   ├── 结果反馈页
│   │   ├── 显示正确角度
│   │   ├── 动画标注：在目标球上标出对应接触点位置（高亮闪烁）
│   │   ├── 显示误差（如「你答了35°，实际是30°，误差5°」）
│   │   └── 「继续下一题」
│   ├── 测试设置（可选）
│   │   ├── 袋型偏好（全部 / 主练角袋 / 主练中袋）
│   │   └── 自适应权重开关（默认开启）
│
├── 进球点对照表（Reference Chart）
│   ├── 交互式角度滑块（0°–90°）
│   │   ├── 拖动滑块 → 目标球动态更新接触点位置
│   │   └── 数字显示：当前角度 + 接触点偏移（距球心 X%）
│   ├── 完整对照表（固定表格）
│   │   ├── 常用角度列表（5°/10°/15°/20°/25°/30°/35°/40°/45°/60°/75°/90°）
│   │   ├── 每行：角度 + 接触点示意图 + 偏移百分比 + 通称（如「1/2球」）
│   │   └── 公式说明：偏移量 = sin(α) × R
│   └── 学习说明
│       ├── 角度瞄准法原理解释
│       └── 如何将对照表用于实战瞄准
│
└── 历史记录（测试成绩追踪）
    ├── 历次测试平均误差趋势图（角袋/中袋分别展示）
    ├── 各角度区间热力图（哪些角度最容易答错）
    └── 最近N次测试记录
```

**角度测试交互细节（含自适应权重逻辑）：**

```
进入角度测试
  → 系统按自适应权重选题（见下方算法）
    → 显示俯视图：目标球（含袋口方向标记）+ 母球位置 + 袋型标注（角袋/中袋）
      → 用户目测角度，数字键盘输入估计值（整数，0–90°）
        → [确认]
          → 结果动画：
            1. 正确击球线高亮（从母球到目标球的实际击球线）
            2. 目标球面上接触点闪烁标注（偏移 = sin(α) × R）
            3. 显示幽灵球位置（可选，帮助理解）
          → 误差提示：「你答了35°，实际是32°，误差3°」
          → 判定结果（专业容差±3°）：✓ 正确 / ✗ 差X度
          → [下一题]
```

**自适应权重算法设计：**

```
出题权重 = 基础权重（30%）+ 误差权重（70%）

误差权重计算：
- 对每个角度区间（以5°为单位，0–5°/5–10°/.../85–90°）
- 记录最近10次该区间的平均误差
- 误差越大的区间，出题概率越高

角袋 vs 中袋分类：
- 角袋（底袋）：典型切入角范围 0°–75°
- 中袋：典型切入角范围 15°–60°（受台面几何约束）
- 每次出题在两种袋型间平衡（大约 60% 角袋 / 40% 中袋）
- 分开统计角袋和中袋的误差，分别做自适应

推荐每次训练题量：20题（约5–8分钟）

难度模式（仅一档，专业）：
- 容差 ±3°（答对：在正确答案±3°以内）
- 角度精度：实际题目以5°为整数精度生成（5/10/15/20...）

历史误差跟踪（对应历史记录功能）：
- 以5题为一组统计平均误差
- 历史图表显示误差随时间的下降趋势
- 分角袋/中袋分别展示（雷达图或折线）
```

**进球点对照表 — 核心数据（预计算）：**

| 切入角 | sin(α) | 接触点偏移 | 通称 |
|---|---|---|---|
| 0° | 0.00 | 球心（满球） | 全球 |
| 10° | 0.17 | 17% 半径 | — |
| 15° | 0.26 | 26% 半径 | — |
| 20° | 0.34 | 34% 半径 | — |
| 25° | 0.42 | 42% 半径 | — |
| 30° | 0.50 | 50% 半径（球心至边缘1/2） | 二分之一球 |
| 35° | 0.57 | 57% 半径 | — |
| 40° | 0.64 | 64% 半径 | — |
| 45° | 0.71 | 71% 半径 | — |
| 48.6° | 0.75 | 75% 半径 | 四分之三点 |
| 60° | 0.87 | 87% 半径 | — |
| 75° | 0.97 | 97% 半径 | — |
| 90° | 1.00 | 球边缘 | 极薄球 |

## 4. 核心用户路径

### 路径一：「今天去球馆，按计划练球」（高频场景）

```
打开 App（训练 Tab）
  → 今日训练卡（看今天练什么）
    → [开始训练]
      → Drill 1 卡片：示意图 + 要点
        → 记录每组进球数（大按钮快速点击）
          → 完成 Drill 1 → 自动进入下一个
            → ... 完成所有 Drill
              → 训练完成页
                → 可选：写今日心得
                  → 保存 ✓
```

**目标：从打开 App 到开始第一个 Drill，不超过 3 步。**

### 路径二：「想查一个技术动作」（中频场景）

```
动作库 Tab
  → 选分类（如「走位」）
    → 选子类（如「一库走位」）
      → 找到目标 Drill
        → 查看示意图 + 看教学视频 + 阅读要点
          → [加入下次训练计划] 或直接关闭
```

### 路径四：「专项训练角度感知」（中频场景）

```
角度 Tab
  → [开始测试]
    → 看球台图，目测切入角
      → 数字键盘输入估计角度
        → 确认 → 动画反馈（正确角度 + 接触点标注）
          → 误差提示 → [下一题]
（可连续练习，累积历史误差统计）

角度 Tab → 对照表
  → 拖动滑块选择角度
    → 实时更新目标球接触点动画
      → 查看完整对照表
```

```
历史 Tab → 统计视图
  → 看本月训练天数/时长概览
    → 点击某个 Drill 查成功率趋势
      → 发现薄弱项
        → 跳转到动作库查看详情
          → 加入下次计划
```

## 6. 设计原则

- **练球时手机使用场景**：球馆光线可能不足，文字应足够大；主操作按钮大而易点（44pt以上）
- **单手操作优先**：底部 Tab + 关键操作按钮置于屏幕下方，符合拇指热区
- **减少打断**：训练中记录流程不强制输入文字，进球数用数字大按钮快速点击
- **HIG 一致性**：使用系统字体、SF Symbols、标准 Sheet/NavigationStack 模式
- **离线优先**：动作库内容和训练记录功能完全可离线使用，同步在后台进行（CloudKit 支持）
