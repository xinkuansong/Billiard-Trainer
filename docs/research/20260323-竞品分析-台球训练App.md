## 国际市场（iOS App Store 美区）

搜索关键词：「billiard training」「pool trainer」「snooker training」

### DrillRoom: AI Pool Trainer ⭐ 最重要竞品

- **链接**：https://drillroom.ai / App Store id1539827505
- **评分**：4.6/5（242条评价，美区）；3.9/5（9条，英区）
- **定价**：免费下载 + 订阅制（解锁50+额外drill及高级功能）
- **最新版本**：3.0.4，更新于2025年10月

**核心功能：**
- 80+ 结构化训练 drill（与职业选手合作设计，包括 Dr. Dave Billiards、Jennifer Barretta、Ralph Eckert、David Muñoz）
- **AI + AR 实时分析**：手机摄像头俯拍球桌（需三脚架，距桌面5英尺以上），自动识别击球轨迹、母球/目标球运动路径
- 实时语音反馈（职业教练声音）
- 击球统计：进球率、球速统计、热力图（台面不同位置准确率）
- 自定义 drill 编辑器：可设计个人训练场景
- Live Capture：把真实球桌情况转为可练习的 drill
- 社区：用户分享 drill
- Apple Watch 伴侣 App（远程控制训练）
- 多用户档案（订阅功能）

**设置要求**：iPhone/iPad + 三脚架，好光线；无外设需求

### WPB: Pool Training & Drills

- **定位**：结构化 drill 课程体系
- **功能**：200+ 结构化 drill，逐步进阶课程，母球可视化，break 速度计算器，进度追踪
- **定价**：免费7天试用 + 订阅（月付/年付/终身）
- **特点**：内容最全的 drill 库，但无 AI 分析

### ChalkySticks Trainer

- **评分**：3.9/5（8条）
- **功能**：8球/9球/10球训练，多难度 drill，自定义图解
- **定价**：免费

### Billiard Shot Studio

- **评分**：5.0/5（10条，样本小）
- **功能**：球桌模拟器，Shot Editor 练习瞄准，Shot Library
- **定价**：免费 + 内购

## 关键结论（写入 docs/03-竞品分析.md）

1. **中国市场完全空白**：验证了训练工具型台球App的市场机会
2. **DrillRoom 是最强国际对标**：AI摄像分析是其核心护城河，但无中文、不含斯诺克/中式台球、无系统知识内容
3. **差异化可行路径**：中文化 + 斯诺克/中式台球内容 + 训练日志+心得记录 + 知识库（DrillRoom几乎没有理论内容）
4. **AI摄像分析勿在v1.0强求**：DrillRoom花了数年打磨，是竞争护城河，但也是高复杂度功能，适合v2.0
