# 调研：台球训练内容体系 + 动作库分类

**日期**：2026-03-23
**来源**：新锐台球学院（buole.com）、知乎/搜狐台球教学、Dr. Dave Pool Info、DrillRoom 功能页

---

## 一、参考分级体系（来源：新锐台球学院）

| 等级 | 对应球龄 | 核心训练内容 |
|---|---|---|
| 入门级（业余一级） | 0–6个月 | 基本动作、简单进球、底袋/中袋角度训练 |
| 初级（业余二级） | 6–18个月 | 白球打点、简单走位、斯登杆法、初级蛇彩 |
| 中级（业余三级） | 12–24个月 | 中远台准度、加塞训练、中级蛇彩 |
| 高级/职业（业余四级+） | 24个月以上 | 高级蛇彩、复杂走位、实战球形 |

## 二、完整训练内容分类树（「台球动作库」骨架）

### 大类 1：基础功（Foundation）

适用球种：通用（中式台球 + 9球）

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 持杆姿势 | 握杆方式（开放式/封闭式）、力度控制 | 入门 |
| 站位姿势 | 脚步、重心、俯身角度 | 入门 |
| 架杆类型 | 标准架、高架、单指架等 | 入门 |
| 出杆动作 | 后摆幅度、加速点、跟杆延伸 | 入门→初级 |
| 瞄准基础 | 三点一线、重合比例法（1/4球/1/2球/3/4球） | 入门 |
| 节奏与预备 | 击球前例行动作、心理稳定性 | 入门→初级 |

---

### 大类 2：准度训练（Pocketing Accuracy）

适用球种：通用

| 训练项目 | 具体说明 | 组数建议 | 推荐级别 |
|---|---|---|---|
| 直线球入袋 | 白球与目标球成直线，近/中/远台 | 每组10–15球 | 入门 |
| 五分点（1/2球）训练 | 目标球置球台中心点，各角度入袋 | 每组15球 | 入门→初级 |
| 三分点（1/4球）训练 | 薄球练习，各角度 | 每组15球 | 初级 |
| 七分点（3/4球）训练 | 厚球练习，各角度 | 每组15球 | 初级 |
| 中袋角度系列 | 底袋到中袋各标准角度（30°/45°/60°） | 每组10球 | 入门→初级 |
| 底袋角度系列 | 底袋各标准角度系列 | 每组10球 | 入门 |
| 远台准度训练 | 长台精准入袋（2/3台以上距离） | 每组10球 | 中级 |
| 侧面角度系列 | 从侧库到各袋口的标准角度 | 每组10球 | 初级→中级 |

---

### 大类 3：杆法（Cue Ball Spin / Stroke Technique）

适用球种：通用

#### 3.1 纵向旋转杆法

| 训练项目 | 核心原理 | 组数建议 | 推荐级别 |
|---|---|---|---|
| 中杆（定杆）练习 | 击打母球赤道，碰球后停止 | 每组15球 | 入门 |
| 高杆（跟杆）练习 | 击打母球上方，碰球后前进 | 每组15球 | 入门→初级 |
| 低杆（缩杆）练习 | 击打母球下方，碰球后后退 | 每组15球 | 初级 |
| 斯登杆法 | 出杆延伸控制，用于精确分离角 | 每组10球 | 初级→中级 |
| 强低杆练习 | 大力低杆，长距离后退 | 每组10球 | 中级 |

#### 3.2 横向旋转（加塞）

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 左塞基础 | 左侧旋转基础控制，认识让点偏移 | 中级 |
| 右塞基础 | 右侧旋转基础控制 | 中级 |
| 顺塞走位 | 沿库边方向旋转，吃库后继续前进 | 中级 |
| 反塞走位 | 反向旋转，吃库后折回 | 中级 |
| 加塞让点补偿 | 力度/旋转量与让点关系实战练习 | 中级→高级 |

---

### 大类 4：分离角控制（Separation Angle）

适用球种：通用（核心技能）

| 训练项目 | 核心原理 | 推荐级别 |
|---|---|---|
| 90度规则（中杆） | 中杆碰球后母球与目标球成90°分离 | 初级→中级 |
| 低杆分离角系列 | 分离角 > 90°，练习各角度球下的低杆分离 | 中级 |
| 高杆分离角系列 | 分离角 < 90°，练习各角度球下的高杆分离 | 中级 |
| 斯登分离角控制 | 利用斯登杆法精确控制分离角 | 中级→高级 |
| 不同角度球分离角实战 | 30°/45°/60°/75°角度球各杆法下分离角练习 | 中级→高级 |

---

### 大类 5：走位训练（Position Play）

适用球种：通用（这是训练体系的核心）

#### 5.1 不吃库走位

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 入位球基础 | 将母球定向走到指定区域（不吃库） | 初级 |
| 短距离走位控制 | 母球行进10–30cm，精确停球 | 初级 |
| 中台走位路线系列 | 10种以上不吃库走位线路练习 | 初级→中级 |

#### 5.2 一库走位

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 一库走位基础 | 母球吃一面库后到达目标区域 | 初级→中级 |
| 高杆一库走位 | 结合高杆的一库走位路线 | 中级 |
| 低杆一库走位 | 结合低杆的一库走位路线 | 中级 |
| 加塞一库走位 | 结合加塞的一库折返走位 | 中级→高级 |

#### 5.3 多库走位

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 两库走位路线 | 母球经过两面库的走位路线控制 | 中级→高级 |
| 多库加塞走位 | 结合加塞旋转的多库走位 | 高级 |
| 对角走位系列 | 从对角球形走位到中台控制 | 高级 |

---

### 大类 6：控力（Speed Control）

适用球种：通用

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 轻推停球系列 | 极小力度控制母球停在指定位置 | 初级 |
| 中力均匀出杆 | 标准力度下各杆法的力量一致性 | 入门→初级 |
| 力度梯度练习 | 同一线路下1–5档力度的母球停点分布 | 中级 |
| 长台速度控制 | 远台击球的力量与到位精度 | 中级 |

---

### 大类 7：特殊球路（Special Shots）

适用球种：通用

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 翻袋（反袋）训练 | 目标球借库反弹入袋，镜像原理练习 | 初级→中级 |
| 翻中袋系列 | 专项中袋翻袋角度练习（初学重点） | 初级 |
| 翻底袋系列 | 底袋翻袋各角度练习 | 中级 |
| K球（贴球碰袋）- 不吃库 | 母球借目标球折射进球（不吃库） | 中级 |
| K球（贴球碰袋）- 吃一库 | 母球借目标球折射后吃库进球 | 中级→高级 |
| 贴库球处理 | 目标球紧贴库边的进球技巧 | 中级 |
| 安全球基础 | 主动布置困难局面给对手 | 中级→高级 |

---

### 大类 8：综合训练球形（Pattern Drills）

适用球种：通用（这是综合能力的检验）

#### 8.1 蛇彩系列（最经典综合训练）

| 训练项目 | 球形说明 | 推荐级别 |
|---|---|---|
| 初级蛇彩（横向） | 15颗球沿台面横向中线排列 | 初级 |
| 初级蛇彩（纵向） | 15颗球沿台面纵向中线排列 | 初级 |
| 双排蛇彩 | 两排球形，综合走位 | 中级 |
| 十字蛇彩 | 球形排成十字，难度提升 | 中级 |
| L型蛇彩 | L形球形 | 中级 |
| 环形蛇彩 | 球形排成环状 | 高级 |
| 贴库综合蛇彩 | 球形贴库放置，难度最高 | 高级 |

#### 8.2 九球标准球形

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 9球清台练习 | 按1–9球顺序击打完成走位 | 中级 |
| 9球最优路线 | 寻找最省力的清台路线 | 高级 |

#### 8.3 进阶综合球形

| 训练项目 | 说明 | 推荐级别 |
|---|---|---|
| 直线球组合走位 | 多个直线球连续走位 | 初级→中级 |
| 贴库球综合（15球） | 大量贴库球综合处理 | 中级→高级 |
| 艾伦式进阶球形系列 | 顶库5球、X形、Y形、特殊球形等 | 高级（建议单杆40+后尝试） |

## 三、DrillRoom 分类体系（国际参考）

DrillRoom 的5个训练大类：

| 大类（英文） | 中文对应 | 我方体系对应 |
|---|---|---|
| Speed Control | 速度/控力 | 大类 6：控力 |
| Shotmaking | 准度/进球 | 大类 2：准度训练 |
| Cue Direction | 母球方向/杆法 | 大类 3：杆法 + 大类 4：分离角 |
| Positional Play | 走位 | 大类 5：走位 |
| Safety | 安全球/战术 | 大类 7 部分 |

**Dr. Dave 体系补充**（国际主流）：
- Wagon Wheel Drill（车轮钻）：以袋口为中心，8个方向的中杆精度测试
- Line-of-balls Drill：一排球走位连打
- Stop/Follow/Draw Drill：同一球位三种杆法对比练习
- No-rail Pattern Play：不吃库的走位路线规划练习
- Runout Drill System（RDS）：清台路线系统训练

## 四、训练项目数据模型（每个Drill的属性）

```
训练项目（Drill）
├── 名称（中文名 + 可选英文）
├── 球种标签（中式台球 / 9球 / 通用）
├── 技术大类（基础功/准度/杆法/分离角/走位/控力/特殊球路/综合球形）
├── 技术子类
├── 难度级别（入门/初级/中级/高级）
├── 球台示意图（标注母球位置/目标球位置/目标袋口/建议走位路线）
├── 文字说明（技术要点、常见错误、练习提示）
├── 教学视频（可链接外部或内嵌）
├── 标准组数（例：每组15球）
├── 参考完成标准
│   ├── 入门达标：15球进6球
│   ├── 初级达标：15球进9球
│   ├── 中级达标：15球进12球
│   └── 高级达标：15球进14球以上
├── 相关训练项目（推荐关联Drill）
└── 收藏/已掌握标记
```

## 五、推荐初期内容量（MVP动作库）

| 大类 | 建议初期Drill数量 | 备注 |
|---|---|---|
| 基础功 | 6项 | 图文为主，无需太多 |
| 准度训练 | 8–10项 | 核心高频使用 |
| 杆法 | 8–10项 | 纵向5项+横向5项 |
| 分离角 | 5–6项 | 难点，需图解清晰 |
| 走位 | 12–15项 | 最重要，内容最多 |
| 控力 | 4–5项 | 可先简化 |
| 特殊球路 | 6–8项 | 翻袋+K球为重点 |
| 综合球形 | 8–10项 | 蛇彩系列为核心 |
| **合计** | **约 57–70项** | 覆盖入门到高级 |
