## 1. 产品概述

**一句话定位**：面向中式八球和九球爱好者的台球训练记录与角度感知学习工具。

**目标用户**：
- 台球爱好者（入门到高级），日常在球馆练球
- 主打中式八球和九球两大球种

**核心价值主张**：
- **动作库**：72 个台球训练项目（Drill），含球台动画图解、分级教学
- **训练记录**：像健身 App 记录动作一样记录每次练球（组数 × 每组球数）
- **角度训练**：独创角度感知测试 + 进球点对照表，帮助建立角度直觉
- **数据复盘**：日历回顾 + 成功率趋势统计

**概念映射**（参照健身训练 App「训记」）：

| 健身 App 概念 | 球迹对应 |
|---|---|
| 健身动作（如杠铃卧推） | 台球训练项目 Drill（如五分点定杆练习） |
| 动作分类（胸/背/腿） | 训练分类（准度/杆法/走位，共 8 大类） |
| 训练计划（胸肌日） | 台球训练计划（今日专项走位训练） |
| 组数 × 重复次数 | 组数 × 每组球数（如 3 组 × 15 球） |
| 进度（力量增长） | 进度（完成率/成功率提升） |

**技术平台**：iOS 17+，SwiftUI 原生，支持 Light/Dark Mode。

## 2. 全局导航与信息架构

### 2.1 底部 Tab 导航

App 使用 5 Tab 底部导航栏，每个 Tab 内使用独立的 NavigationStack（导航深度一般 2-3 层）。

| Tab 序号 | 名称 | SF Symbol 图标 | 使用频率 | 定位 |
|---|---|---|---|---|
| 1 | 训练 | `dumbbell.fill` | 每次练球必用 | 日常最高频入口 |
| 2 | 动作库 | `list.bullet.rectangle` | 中频 | 查资料、选 Drill |
| 3 | 角度 | `angle` | 中频 | 专项角度感知训练 |
| 4 | 历史 | `clock.arrow.circlepath` | 低频 | 复盘与统计 |
| 5 | 我的 | `person.circle` | 低频 | 个人信息与设置 |

### 2.2 首次启动流程

```
App 启动
  → 判断是否完成引导
    → 未引导：显示 Onboarding（3 页引导）→ 可选登录 → MainTabView
    → 已引导：直接进入 MainTabView（默认训练 Tab）
```

### 2.3 完整页面层级树

```
App
├── Onboarding（首次启动引导，3 页）
│   └── LoginView（登录选项）
│       └── PhoneLoginView（手机验证码登录）
│
└── MainTabView（5 Tab）
    │
    ├── Tab 1：训练
    │   └── TrainingHomeView（今日训练首页）
    │       ├── → PlanListView（训练计划列表）
    │       │   ├── → PlanDetailView（官方计划详情）
    │       │   └── → CustomPlanBuilderView（新建/编辑自定义计划）
    │       └── → ActiveTrainingView（全屏训练记录流）★
    │           ├── 内嵌 DrillPickerSheet（选择训练项目弹窗）
    │           ├── → TrainingNoteView（训练心得）
    │           └── → TrainingSummaryView（训练总结）
    │
    ├── Tab 2：动作库
    │   └── DrillListView（动作列表 + 筛选）
    │       └── → DrillDetailView（动作详情）
    │
    ├── Tab 3：角度
    │   └── AngleHomeView（角度训练首页）
    │       ├── → AngleTestView（角度测试答题）★
    │       ├── → ContactPointTableView（进球点对照表）
    │       └── → AngleHistoryView（测试历史统计）
    │
    ├── Tab 4：历史
    │   └── HistoryCalendarView（月历视图）
    │       ├── → TrainingDetailView（单次训练详情）
    │       └── → StatisticsView（统计图表）
    │
    └── Tab 5：我的
        └── ProfileView（个人中心）
            ├── → FavoriteDrillsView（收藏列表）
            │   └── → DrillDetailView（动作详情）
            └── → LoginView（登录）
                └── → PhoneLoginView（手机登录）
```

> ★ 标记的是产品核心交互页面，需重点设计。

## 3. 逐屏功能清单

### Tab 1 — 训练（Training）

---

#### 3.1.1 TrainingHomeView — 今日训练首页

**文件路径**：`QiuJi/Features/Training/Views/TrainingHomeView.swift`

**功能描述**：训练 Tab 的根页面，展示当前激活计划的今日训练安排，引导用户快速开始训练。

**页面元素**：
- **导航栏**：大标题「训练」，右上角 toolbar 按钮「训练计划」（跳转 PlanListView）
- **有激活计划时**：
  - 计划头部卡片：计划名称、当前进度（第 X 天 / 共 Y 天）
  - 环形进度指示器：今日完成进度
  - 今日 Drill 列表：每项显示名称、分类、目标组数
  - 底部主按钮「开始训练」→ 全屏进入 ActiveTrainingView
- **无激活计划时（空状态）**：
  - BTEmptyState 组件：图标 + 文案引导
  - 两个引导按钮：「选择训练计划」/「自由记录」

**交互说明**：
- 点击「开始训练」→ `fullScreenCover` 弹出 ActiveTrainingView
- 点击「训练计划」→ push 到 PlanListView
- 下拉可刷新

**空状态**：无激活计划时显示引导选择计划或自由记录
**Freemium 限制**：无

#### 3.1.2 PlanListView — 训练计划列表

**文件路径**：`QiuJi/Features/Training/Views/PlanListView.swift`

**功能描述**：展示所有可用的训练计划，分为官方计划和自定义计划。

**页面元素**：
- **导航栏**：标题「训练计划」，右上角「新建」按钮（创建自定义计划）
- **官方计划区**：
  - 按等级分组（入门 / 初级 / 中级 / 高级 / 综合）
  - 每套计划卡片：名称、等级标签、周期说明、Pro 锁标识（付费计划）
  - 共 6 套计划
- **自定义计划区**：
  - 用户创建的计划列表
  - 支持滑动删除（带确认 Alert）

**交互说明**：
- 点击官方计划 → push 到 PlanDetailView
- 点击「新建」→ push 到 CustomPlanBuilderView
- 长按/滑动自定义计划 → 删除确认弹窗

**空状态**：自定义计划区为空时可不显示该分区
**Freemium 限制**：
- 免费计划：新手入门 8 周计划、基础杆法专项（2 套）
- Pro 计划（显示锁标识）：走位突破、中级综合突破、高级加塞与多库、全能综合计划（4 套）

**文件路径**：`QiuJi/Features/Training/Views/PlanDetailView.swift`

#### 3.1.4 CustomPlanBuilderView — 自定义计划创建

**文件路径**：`QiuJi/Features/Training/Views/CustomPlanBuilderView.swift`

**功能描述**：创建或编辑用户自定义训练计划。

**页面元素**：
- **表单**（List insetGrouped 样式）：
  - 计划名称输入框（TextField）
  - 每周训练天数设置（Stepper：1-7 天）
- **训练项目列表**：
  - 已添加的 Drill 列表
  - 每项显示：Drill 名称、目标组数、每组球数

- 点击「添加」→ Sheet 弹出 Drill 选择列表

#### 3.1.5 ActiveTrainingView — 全屏训练记录流 ★

**文件路径**：`QiuJi/Features/Training/Views/ActiveTrainingView.swift`

**功能描述**：训练进行中的核心记录界面，全屏模式，含计时、Drill 切换、球数记录。这是产品最重要的交互页面。

**页面元素**：
- **顶部导航**：内嵌 NavigationStack，左上角关闭/结束按钮
- **计时器**：训练时长实时显示
- **Drill 分页切换**：TabView 分页样式，左右滑动切换不同 Drill
- **当前 Drill 区域**：
  - Drill 名称 + 分类标签
  - **BTBilliardTable 球台动画**：Canvas 绘制的球台 + 母球/目标球路线动画
  - 训练要点文字提示
- **球数记录区**：
  - 大号进球计数器（当前组进球数 / 总球数）
  - 加减按钮（大尺寸，方便单手操作）
  - 「完成本组」按钮
  - 分组进度网格（显示每组完成情况）
- **底部操作**：
  - 「添加项目」→ Sheet 弹出 DrillPickerSheet
  - 「结束训练」→ Alert 确认
- **DrillPickerSheet**（内嵌弹窗）：从 Bundle 加载的 Drill 列表，可选择添加到本次训练

**交互流程**：
```
开始训练（全屏）
  → 当前 Drill 页面（球台动画 + 要点）
    → 记录本组：大按钮计数「进了 X / 共 Y 球」
      → 「完成本组」→ 下一组 / 本 Drill 全部完成
        → 左右滑动切换到下一个 Drill
          → 所有 Drill 完成 / 手动结束
            → TrainingNoteView（心得页）
              → TrainingSummaryView（总结页）
```

**空状态**：无 Drill 时显示添加引导
**Freemium 限制**：无（训练记录对所有用户开放）

#### 3.3.2 AngleTestView — 角度测试答题 ★

**功能描述**：核心角度感知训练功能。随机生成球台布局，用户目测切入角并输入估计值。

**页面元素**：
- **顶部进度条**：当前题号 / 总题数
- **球台 Canvas 区域**（BTAngleTestTable）：
  - 俯视标准球台（6 个袋口可见）
  - 目标球 + 袋口方向标记箭头
  - 母球位置（每题随机生成）
  - 切入角范围：5°-85°，以 5° 为步进
- **输入区域**：
  - TextField 数字键盘输入估计角度（0-90°整数）
  - 主按钮「确认」（BTButtonStyle.primary）
- **结果反馈（确认后动画展示）**：
  - 正确击球线高亮动画（从母球到目标球）
  - 目标球面接触点闪烁标注
  - 可选幽灵球位置显示
  - 误差文案：「你答了 35°，实际是 30°，误差 5°」
  - 判定：正确（误差 ≤ ±3°）/ 不正确
  - 结果气泡 + 颜色反馈
- **下一题按钮**
- **每轮结束总结**：正确率、平均误差

**自适应出题逻辑**（影响 UI 展示）：
- 出题权重 = 基础权重 30% + 误差权重 70%
- 误差大的角度区间出题概率更高
- 角袋约 60% / 中袋约 40%
- 推荐每轮 20 题（约 5-8 分钟）
- 判定容差：±3°

**交互流程**：
```
进入角度测试
  → 显示球台：目标球 + 袋口标记 + 母球
    → 目测角度 → 键盘输入数值 → 「确认」
      → 动画反馈：击球线高亮 + 接触点闪烁 + 误差提示
        → 「下一题」→ 循环
          → 完成一轮 → 总结页
```

**Freemium 限制**：免费每天 20 次测试，Pro 无限

## 7. 设计约束与建议

### 7.1 使用场景约束

- **练球场景**：用户在球馆使用 App，环境光线可能不足
- **单手操作为主**：一只手拿手机记录，另一只手拿球杆
- **时间碎片化**：每组球间隙记录，需要极快的操作速度

### 7.2 交互设计原则

1. **3 步到达核心操作**：打开 App → 今日卡 → 开始训练 → 第一个 Drill 记录界面
2. **按钮最小 44pt**：所有可点击目标满足 HIG 最低触控尺寸
3. **底部操作优先**：主操作按钮置于屏幕下方拇指热区
4. **减少文字输入**：训练记录中不强制输入文字，进球数用大按钮快速点击/加减
5. **离线优先**：所有核心功能（动作库、训练记录、角度训练）可完全离线使用

### 7.3 视觉设计要求

1. **Light / Dark Mode 双模式**：所有页面必须适配
2. **HIG 一致性**：遵循 Apple Human Interface Guidelines
   - 使用 SF Symbols 图标体系
   - 使用系统字体（SF Pro / SF Pro Rounded）
   - 标准 Sheet / NavigationStack / Alert 交互模式
3. **品牌色**：当前主色为绿色（`#1A6B3C` / `#25A25A`），与台球桌绿色台面呼应
4. **球台动画规范**（待设计出稿）：
   - 俯视视角
   - 母球白色、目标球橙色、台面绿色、库边棕色
   - 路径动画多阶段时序：击球前 → 接触 → 母球走位 → 目标球走位
   - 接触点视觉强调（闪烁或缩放）
5. **角度测试结果动画**：
   - 正确击球线渐显高亮
   - 目标球接触点闪烁标注
   - 幽灵球位置渐显（可选功能）

### 7.4 页面状态覆盖清单

每个页面需设计以下状态（如适用）：

## 附录

### A. 核心用户路径

**路径一：按计划练球（每次练球必用）**
```
打开 App → 今日训练卡 → 「开始训练」
  → Drill 卡片（球台动画 + 要点）
    → 记录进球数（大按钮）→ 「完成本组」
      → 下一组/下一项 → 全部完成
        → 心得（可跳过）→ 训练总结 → 保存
```

**路径二：查技术动作**
```
动作库 Tab → 选分类 → 选子类 → Drill 详情
  → 查看动画 + 教学 → 加入训练计划
```

**路径三：角度训练**
```
角度 Tab → 「角度测试」→ 看球台图 → 输入角度 → 确认
  → 结果动画（击球线 + 接触点）→ 下一题 → 循环
```

**路径四：复盘统计**
```
历史 Tab → 日历选日期 → 训练详情 → 统计图表
  → 发现薄弱项 → 跳转动作库 → 加入计划
```
