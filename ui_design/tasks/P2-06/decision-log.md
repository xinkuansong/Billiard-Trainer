# 设计决策日志 — P2-06

## 决策 1: 付费墙为全屏深色独立页面
- **上下文**: SubscriptionView 从多个入口（ProfileView 订阅管理、各 Pro 锁入口）进入，需确认展示模式
- **决策**: 全屏深色页面（#111111 背景），左上角 ✕ 关闭按钮，无导航栏无 Tab 栏
- **理由**: 设计文档 4.5 明确为全屏深色主题。深色背景创造沉浸式高端感，与 App 日常浅色 UI 形成反差，强化"Pro 专属"心理暗示。与 P2-03 访客态 Pro 推广卡（深色 #1C1C1E）的视觉基调一脉相承
- **日期**: 2026-04-05
- **影响范围**: 无需 Dark Mode 变体（自身已是深色主题，状态矩阵已标注）

## 决策 2: 品牌文案精简为 1 行
- **上下文**: prompt 中指定 3 行品牌文案，但 v1 因内容过多导致方案卡片被挤出视口
- **选项**: A) 保持 3 行文案，大幅压缩功能列表间距; B) 精简为 1 行文案，保持功能列表和方案卡片的可读性
- **决策**: 采用方案 B — 1 行文案 "专为台球爱好者设计的专业训练系统"
- **理由**: 付费墙的核心转化目标是：功能说服→价格展示→CTA 点击。品牌文案是辅助元素，1 行足以传达定位。将宝贵的屏幕空间让给 6 项功能和 3 档定价卡片，确保核心转化链路一屏完整可见
- **日期**: 2026-04-05

## 决策 3: 年订阅卡视觉最突出（推荐标签+绿色边框+勾选+月均标注）
- **上下文**: 3 档方案需要明确的视觉层级引导用户选择推荐档位
- **决策**: 年订阅 ¥88 默认选中，使用品牌绿 #1A6B3C 边框 + 顶部 "推荐" pill badge + 右上角绿色勾选 + "月均¥7.3" 绿色性价比标注
- **理由**: 年订阅是最优 LTV 档位（月均 ¥7.3 vs 月订阅 ¥18），视觉引导应最强。绿色=推荐选中，金色=终身买断的特殊性，灰色=月订阅的基础性，三色三档层级清晰
- **日期**: 2026-04-05
- **影响范围**: E-04 Dark Mode 无需额外适配（本页已是深色）
