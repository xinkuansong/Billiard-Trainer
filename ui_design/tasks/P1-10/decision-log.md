# 设计决策日志 — P1-10

## 决策 1: 统计页 Pro 锁使用「卡片剪影 + 渐变磨砂 + 居中锁区」布局
- **上下文**: 设计文档指定统计图表为全遮罩锁定，需决定具体的视觉呈现方式
- **选项**: A) 全页面均匀半透明遮罩; B) 渐变磨砂 + 卡片剪影 + 居中锁区分层
- **决策**: 采用方案 B — 上部轻磨砂露出卡片/图表剪影，中部居中锁图标+文案+CTA，下部卡片轮廓暗示更多内容
- **理由**: 渐变分层比均匀遮罩更有层次感和转化力，用户能感知到「丰富数据被锁住」而非「空白页被锁住」
- **日期**: 2026-04-04

## 决策 2: 全遮罩锁区 CTA 使用金色填充按钮，延续 A-04 决策
- **上下文**: A-04 确立渐进式锁用描边按钮、全遮罩用填充按钮的差异化
- **决策**: 「解锁 Pro」按钮为金色 #D4941A 填充胶囊按钮，与 P1-04 底部金色 CTA 一致
- **理由**: 统计页是全遮罩场景（非渐进式），应使用强 CTA 填充按钮。跨页面 Pro 锁视觉系统保持一致
- **日期**: 2026-04-04

## 决策 3: 锁图标圆形琥珀容器延续 P1-04 模式
- **上下文**: P1-04 确立了锁图标使用 #FFDDAF 浅琥珀圆形容器 + #D4941A 深金图标
- **决策**: 统计页锁图标完全沿用此模式
- **理由**: Pro 锁视觉系统统一，用户在不同页面看到相同的锁形态，建立一致的 Premium 认知
- **日期**: 2026-04-04
