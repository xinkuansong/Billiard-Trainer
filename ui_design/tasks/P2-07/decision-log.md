# 设计决策日志 — P2-07

## 决策 1: FavoriteDrillsView 为纯列表子页面，无搜索/筛选
- **上下文**: 收藏列表与 DrillListView 共用 BTDrillCard 组件，需确认是否包含搜索栏和筛选 Chip
- **决策**: 不包含搜索栏和筛选 Chip，纯 LazyVStack 列表
- **理由**: 设计文档 3.2.3 定义为简洁列表页面，收藏数量通常有限（几十个以内），无需搜索/筛选。与 DrillListView 形成功能层级区分：完整库浏览（搜索+筛选+分区）vs 个人收藏（简洁列表）
- **日期**: 2026-04-05
- **影响范围**: 如后续用户收藏量增大，可考虑 v2 添加搜索

## 决策 2: 缩略图沿用 P1-01 的 64pt 照片风格
- **上下文**: A-03 原始 BTDrillCard 规范为纯文字卡片，P1-01 决策 2 增加了 64pt 台球照片缩略图
- **决策**: 收藏列表沿用 P1-01 的照片缩略图模式
- **理由**: 保持与 DrillListView 的卡片一致性，用户在两个页面间切换时视觉连贯
- **日期**: 2026-04-05
