# 设计决策日志 — A-08

## 决策 1: BTBilliardTable 球位使用斜线布局
- **上下文**: v1 中母球和目标球都在垂直中心线上，形成单调的直线路径
- **选项**: A) 垂直居中布局（简洁但无角度感）; B) 斜线对角布局（展示角度击球场景）
- **决策**: 采用斜线对角布局，母球 (38%, 70%)，目标球 (58%, 32%)
- **理由**: 球迹的核心训练内容包含走位和角度练习，斜线路径更能体现组件在真实场景中的表现力
- **日期**: 2026-04-03
