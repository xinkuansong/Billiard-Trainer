# 设计决策日志 — P0-08

## 决策 1: 移除 Stitch 自行添加的装饰元素
- **上下文**: v1 中 Stitch 添加了 "NOTE." 绿色大标题（含下划线）和右上角半透明圆形装饰图形
- **选项**: A) 保留装饰元素增加品牌感 B) 移除保持极简
- **决策**: 移除所有装饰元素
- **理由**: TrainingNoteView 是训练后的心得记录页，核心价值是提供安静的写作空间。装饰元素分散注意力，与参考截图和设计文档的极简定位不符
- **日期**: 2026-04-04

## 决策 2: 保留 "在此开始记录..." 占位符
- **上下文**: Stitch 在文本编辑区添加了灰色占位符 "在此开始记录..."，设计文档未明确要求
- **决策**: 保留该占位符
- **理由**: 占位符是标准 UX 实践，配合上方两行引导提示为用户提供明确的输入指引，降低空白页面的认知负担
- **日期**: 2026-04-04
