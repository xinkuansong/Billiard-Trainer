## 四、用户流程

### 流程 1：浏览 Drill 分类列表

**步骤**：
1. 启动 App → 点击底部「动作库」Tab
2. 向下滚动，确认 8 大分类（基本功、准度、杆法、母球分离、走位、力量控制、特殊球、综合）均有内容
3. 点击任意分类下的一个 BTDrillCard

**预期结果**：进入 DrillDetailView 详情页

- [ ] 8 大分类均可见且有 Drill 条目
- [ ] 点击卡片后正确进入详情页

### 流程 2：搜索 Drill

**步骤**：
1. 在动作库页面顶部搜索框输入「直线」
2. 观察列表实时过滤
3. 清空搜索框
4. 输入一个不存在的关键词（如「zzzzz」）

**预期结果**：
- 输入「直线」后只显示包含「直线」的 Drill
- 输入不存在的关键词后显示空状态

- [ ] 搜索过滤正确生效
- [ ] 空状态视图正常展示
- [ ] 清空搜索后列表恢复完整

### 流程 3：球种筛选

**步骤**：
1. 点击「中式台球」筛选按钮
2. 观察列表只显示中式台球相关 Drill
3. 点击「9球」
4. 点击「全部」恢复

**预期结果**：列表根据球种实时切换

- [ ] 筛选切换后列表立即更新
- [ ] 「全部」显示完整列表

### 流程 4：收藏 Drill 并查看收藏夹

**步骤**：
1. 动作库 Tab → 点击任意免费 Drill 进入详情
2. 点击右上角心形收藏按钮（空心 → 实心）
3. 返回列表
4. 切换到「我的」Tab → 点击「收藏夹」
5. 确认刚收藏的 Drill 出现在列表中
6. 点击进入该 Drill 详情 → 取消收藏（实心 → 空心）
7. 返回收藏夹列表

**预期结果**：收藏/取消收藏实时生效

- [ ] 收藏后心形变为实心
- [ ] 收藏夹中出现该 Drill
- [ ] 取消收藏后从收藏夹消失

### 流程 5：付费 Drill 锁定体验

**步骤**：
1. 在动作库列表中找到一个带锁定图标的付费 Drill
2. 点击进入详情页
3. 确认内容被 BTPremiumLock 遮罩覆盖
4. 点击「解锁全部内容」按钮

**预期结果**：遮罩阻止查看内容，点击按钮触发订阅引导（P7 前为 stub）

- [ ] 付费 Drill 内容被遮罩，无法看到正文
- [ ] 「解锁全部内容」按钮可点击
- [ ] 免费 Drill 无遮罩，内容完整展示
