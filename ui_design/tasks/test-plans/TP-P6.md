## 四、用户流程

### 流程 1：日历浏览与训练详情查看

**步骤**：
1. 启动 App → 历史 Tab
2. 确认当月日历展示，有训练记录的日期有绿色标记
3. 点击一个有记录的日期 → 查看当日训练列表
4. 点击某条训练记录 → 进入训练详情页
5. 确认详情页信息完整（日期、时长、Drill 数据、成功率）
6. 返回日历视图

**预期结果**：日历 → 日期选择 → 训练详情完整流程通畅

- [ ] 日历正确高亮有记录日期
- [ ] 日期列表展示当日所有训练
- [ ] 训练详情页数据完整
- [ ] 返回导航正常

### 流程 2：翻月浏览历史

**步骤**：
1. 在日历视图点击左箭头翻到上个月
2. 确认月份标题更新（如「2026年3月」）
3. 有训练记录的日期仍正确高亮
4. 再点右箭头翻回当月

**预期结果**：翻月操作流畅，数据正确

- [ ] 翻到上月后月份标题正确
- [ ] 上月有记录日期正确高亮
- [ ] 翻回当月数据无异常

### 流程 3：统计图表验证

**步骤**：
1. 进入统计视图
2. 默认「周」范围 → 确认折线图展示最近 7 天数据
3. 切换到「月」→ 折线图展示最近 4 周数据
4. 切换到「年」→ 折线图展示最近 12 个月数据
5. 确认三个数字卡片在每次切换后更新
6. 向下滚动查看各类别成功率条形图

**预期结果**：图表随时间范围正确切换

- [ ] 周视图 7 个数据点
- [ ] 月视图 4 个数据点
- [ ] 年视图 12 个数据点
- [ ] 数字卡片随切换更新
- [ ] 条形图展示练习过的类别

### 流程 4：Freemium 60 天限制验证

**步骤**：
1. 以匿名/免费用户身份使用
2. 查看最近的训练记录 → 应可正常查看
3. 如有 60 天前的训练记录 → 应显示「升级查看全部历史」遮罩
4. 确认遮罩包含订阅引导

**预期结果**：60 天限制正确生效

- [ ] 60 天内记录可正常访问
- [ ] 60 天前记录有遮罩（如有旧数据可测试）
- [ ] 遮罩包含升级引导文字

### 流程 5：空数据状态

**步骤**：
1. 全新安装 App（删除后重新安装）
2. 进入历史 Tab → 查看日历
3. 进入统计视图

**预期结果**：空状态友好展示

- [ ] 日历无任何绿色标记
- [ ] 点击任意日期提示「当天无训练记录」
- [ ] 统计视图展示空状态引导（BTEmptyState + 「开始第一次训练」）
- [ ] 不显示空白图表
