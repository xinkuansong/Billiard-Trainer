## 四、用户流程

### 流程 1：完整角度测试（20 题）

**步骤**：
1. 启动 App → 角度 Tab
2. 点击「开始测试」
3. 球台出现第一题 → 观察母球、目标球、袋口位置
4. 数字键盘输入估计角度（0–90）→ 点击「确认」
5. 查看结果：正确路线、用户路线、误差评级
6. 点击「下一题」→ 重复至第 20 题
7. 查看测试总结

**预期结果**：完整测试流程顺畅，统计数据正确

- [ ] 20 题全部完成，题号从 1 递增到 20
- [ ] 每题提交后显示正确答案与误差
- [ ] 总结页平均误差与各题数据一致
- [ ] 精准次数（≤3°误差）统计正确

### 流程 2：角度计算正确性验证

**步骤**：
1. 进入对照表
2. 将滑块拖到 30°，确认偏移 ≈ 50%
3. 将滑块拖到 45°，确认偏移 ≈ 70.7%
4. 将滑块拖到 0°，确认偏移 = 0%
5. 将滑块拖到 90°，确认偏移 = 100%

**预期结果**：数值与 sin(α) 公式一致

- [ ] 30° 偏移 ≈ 50%
- [ ] 45° 偏移 ≈ 70.7%
- [ ] 0° 偏移 = 0%
- [ ] 90° 偏移 = 100%

### 流程 3：历史记录持久化

**步骤**：
1. 完成流程 1（一轮 20 题测试）
2. 返回角度首页 → 进入历史记录
3. 确认折线图有数据点
4. 强制退出 App → 重新打开
5. 再次进入历史记录

**预期结果**：测试数据持久化成功

- [ ] 历史页面有折线图数据
- [ ] 重启后数据仍存在
- [ ] 角袋/中袋分段切换正确

### 流程 4：每日次数限制（Freemium）

**步骤**：
1. 以匿名用户身份完成一轮 20 题测试
2. 再次点击「开始测试」→ 立即显示达到上限提示
3. 确认提示包含 crown.fill 图标和订阅引导文字

**预期结果**：免费用户每日 20 题限制生效

- [ ] 第 21 题无法开始，显示限制提示
- [ ] 提示包含订阅引导（crown.fill 图标）
- [ ] 提示文字清晰说明原因

### 流程 5：对照表浏览

**步骤**：
1. 角度 Tab → 进入「进球点对照表」
2. 上下滑动浏览 13 个标准角度
3. 拖动顶部滑块从 0° 到 90°
4. 确认球面图随角度变化实时更新

**预期结果**：对照表功能完整

- [ ] 13 个标准角度全部展示
- [ ] 滑块拖动流畅
- [ ] 球面接触点随滑块实时变化
