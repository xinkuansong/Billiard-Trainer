# 设计决策日志 — P1-02

## 决策 1: 空状态图标使用圆形浅色背景
- **上下文**: A-03 BTEmptyState 规范为图标直接显示（无容器），P0-02 也无容器。P1-02 Stitch 自动添加了圆形浅底 (#1A6B3C/5) 包裹图标
- **决策**: 保留圆形浅底作为搜索无结果场景的视觉增强
- **理由**: 搜索无结果与"无数据"空状态含义不同，圆形背景增加了图标的视觉重量，使"无搜索结果"信息更直观。视觉效果良好，不影响一致性
- **日期**: 2026-04-04
- **影响范围**: 其他搜索无结果空状态（如 P2-07 收藏空列表）可沿用此模式
