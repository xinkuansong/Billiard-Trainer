## 💡 建议项（非阻塞）

1. **BTSegmentedTab 激活标签文字颜色**: A-06 确定的样式为激活标签文字 `#1A6B3C`（品牌绿），当前为 `text-on-surface`（近黑色 `#1a1c1f`）。在实际页面中深色文字 + 绿色下划线的组合视觉上也可接受，但与 Phase A 组件规格略有偏差。后续一致性审核时关注。

2. **Tab 栏激活色**: `text-emerald-800` 是 Tailwind 工具色，建议后续统一为精确 `#1A6B3C`。

3. **Tab 栏顶部圆角**: `rounded-t-3xl` 偏大，iOS 原生 Tab 栏无顶部圆角。视觉上尚可接受，后续 Phase 可微调。

4. **区块标题字号**: 当前 17px，设计文档指定 16pt。差异极小（1px），不影响视觉效果。
