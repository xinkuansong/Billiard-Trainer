直线球定点练习

中式台球 准度训练 L1 初学
edit 点击此处输入备注

训练要点

  • 1

    目标球对准袋口中心，确保瞄准线笔直

  • 2

    母球击球点位于中心偏下，确保稳定击打

  • 3

    出杆保持水平，避免翘杆导致偏差

  • 4

    控制力度在 3-4 成力，关注准度而非力量

  • 5

    每次出杆前完成 3 次预备动作

训练维度

准度 85%
力量控制 40%
走位判断 60%
杆法技巧 30%
心理素质 20%
