# 设计决策日志 — P1-03

## 决策 1: 导航栏中间显示中文 Drill 名称
- **上下文**: v1 中 Stitch 使用了英文 "Drill Detail"，需决定中间标题的内容
- **选项**: A) 省略中间标题（标准 iOS push 行为）; B) 显示中文 Drill 名称
- **决策**: 显示中文 Drill 名称「直线球定点练习」，颜色 #000000 黑色
- **理由**: 作为信息密集的详情页，导航栏显示当前 Drill 名称有助于用户确认位置；黑色与 P1-01/P0-01 的文字颜色基准一致
- **日期**: 2026-04-04
- **影响范围**: P1-04（Pro 锁定态）应沿用此导航栏模式

## 决策 2: 操作图标行使用灰色而非品牌绿
- **上下文**: v1 中图标按钮使用 #005129（品牌绿），与主操作按钮竞争注意力
- **决策**: 图标按钮颜色改为 rgba(60,60,67,0.6) 辅助灰色
- **理由**: 辅助功能按钮（要点/历史/图表）不应与底部主操作按钮（加入训练）竞争视觉层级。灰色与 P1-01/P0-01 的辅助图标颜色基准一致
- **日期**: 2026-04-04
- **影响范围**: P1-04 同页面应沿用

## 决策 3: 底部操作栏 darkPill + primary 双按钮模式
- **上下文**: 详情页底部需要关闭和主操作两个按钮
- **决策**: 左侧「✕ 关闭」darkPill (#1C1C1E 深色填充+白字) + 右侧「⊕ 加入训练」primary (#1A6B3C 填充+白字)，右侧 flex 填充剩余宽度
- **理由**: darkPill 关闭按钮为低视觉权重的退出操作，primary 为高视觉权重的主操作，层级清晰。与 A-02 BTButton 组件规范一致
- **日期**: 2026-04-04
- **影响范围**: P1-04 及其他详情页底部操作栏应沿用

## 决策 4: DESIGN.md 偏离延续既有模式
- **上下文**: Stitch DESIGN.md 延续 "Disciplined Athlete" 主题，primary #005129、No-Line Rule、Glassmorphism 等偏离
- **决策**: 延续 A-01 决策 3 — 以 code.html 和 screen.png 渲染效果为准，DESIGN.md 仅作参考
- **理由**: code.html 中 primary 正确为 #1A6B3C，背景正确为 #F2F2F7，球台色全部正确
- **日期**: 2026-04-04
