## ⚠️ 问题项（需修改）

### 问题 1: 底部「加入训练」按钮使用渐变 — 必须改为纯色填充

- **当前**: `bg-gradient-to-br from-[#005129] to-[#1A6B3C]` 渐变填充
- **期望**: 纯色 `#1A6B3C` 填充 + 白字
- **依据**: A-01/A-02 决策 — 品牌色纯色填充，禁止渐变

### 问题 2: 底部「关闭」按钮样式错误 — 应为 darkPill

- **当前**: 浅灰色背景 `#F3F3F8` + 深绿色文字 `#005129`
- **期望**: darkPill 样式 — 深色填充 `#1C1C1E` + 白色文字 + ✕ 图标
- **依据**: tasks.md 验收标准明确要求 darkPill 样式（A-02 BTButton darkPill）

### 问题 3: 导航栏中间标题 "Drill Detail" 为英文且颜色错误

- **当前**: "Drill Detail" 英文，颜色 `#005129`（深绿）
- **期望**: 省略中间标题（iOS push 标准行为，仅保留返回箭头 + 「动作库」），或使用中文 Drill 名称缩写，颜色 #000000 黑色
- **依据**: P1-01 决策 1 — 文字颜色应为黑色；全中文 UI

### 问题 4: 主色 #005129 偏离 — 应为 #1A6B3C

- **当前**: Tailwind config 中 primary 为 `#005129`，导致训练要点编号圆圈、条形图填充、「查看精讲」按钮均为 #005129（过深的绿色）
- **期望**: 全部使用 `#1A6B3C`
- **依据**: A-01 品牌色锚定决策 — 品牌绿恒定为 #1A6B3C
- **影响范围**: 编号数字圆圈、5 条横向条形图填充、「查看精讲」按钮填充色

### 问题 5: BTLevelBadge "L1 初学" 样式错误

- **当前**: 实心绿色填充 + 白色文字（这是 L0 入门 的样式）
- **期望**: L1 初学 应为浅绿色底 `rgba(26,107,60,0.12)` + 绿色文字 `#1A6B3C`
- **依据**: A-03 决策 1/2 — BTLevelBadge 多色方案：L0 实心绿+白字 / L1 绿浅底+绿字

### 问题 6: 操作图标行与标签行布局错误

- **当前**: 3 个圆形图标按钮和标签（中式台球/准度训练/L1 初学）在同一行，左右分布
- **期望**: 分为两行 — 第一行：3 个圆形图标按钮（要点/历史/图表）+ 各自下方有文字标签；第二行：球种胶囊 + 分类胶囊 + BTLevelBadge
- **依据**: 设计文档 3.2.2 描述 "操作图标行（三个圆形图标按钮）" 和 "头部标签行" 为独立的两行
