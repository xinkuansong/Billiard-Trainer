## ⚠️ 问题项（需修改）

### P1: BTFloatingIndicator 位置错误 — 居中而非右对齐（高优先级）

- **期望**: 浮动指示器应**右对齐**，距右边缘 16pt，位于 Tab 栏上方 8pt（设计文档 3.6.7 明确要求"水平右对齐 16pt"）
- **当前**: 使用 `left-1/2 -translate-x-1/2` 水平居中对齐
- **修改**: 将定位改为 `right-4`（16pt from right），移除 `left-1/2 -translate-x-1/2`

### P2: BTRestTimer 按钮布局错误 — 垂直堆叠而非水平并排（中优先级）

- **期望**: "+30s" 和 "完成休息" 两个按钮**水平并排**放置，12pt 间距
- **当前**: 使用 `flex-col`（垂直堆叠），两个全宽按钮上下排列
- **修改**: 改为 `flex-row` 并排布局，按钮各占约一半宽度
- **附加**: "完成" 按钮文字应为 "完成休息"（更完整的行动号召）

### P3: 模拟 Tab 栏只有 3 个图标（低优先级）

- **期望**: 4 个 Tab 图标（球迹有 4 个主 Tab：训练/动作库/历史/我的）
- **当前**: 只有 3 个图标（home/monitoring/person）
- **修改**: 添加第 4 个图标
