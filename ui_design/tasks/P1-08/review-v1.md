## 帧 2: TrainingDetailView（训练详情 Sheet）

### ✅ 通过项

- Sheet 模态弹窗结构正确：拖拽条 (36px × 5px) + 内容区
- 顶部导航：✕ 关闭按钮（灰色圆形背景）+ 「存为模版」绿色文字按钮 — 位置正确
- 头部信息区：训练名「杆法专项训练」20pt Bold + 「设置颜色」绿色描边胶囊
- 统计行横向排列：180 进球 / 15 组 / 42 分钟 / 14:30 时段 — 大数字 + 小标签的数据展示层级清晰
- Drill 卡片 1「直线准度练习」：球台缩略图 (40pt, 绿面 #1B6B3A + 棕色边框 #7B3F00) + 60/90 累计 + 5 组逐行明细（含绿色 ✓ 和休息时间）— 结构完整
- 训练心得区：白色卡片 + 标题 + 心得文字 — 正确
- 底部操作栏：「编辑数据」primary 绿色填充 + 「复制到今天」secondary 描边 + 「更多」方形描边按钮 — 三级层次正确
- BTOverflowMenu 展开态：白色圆角浮层 + 5 项菜单 + 彩色圆形图标背景 — 整体结构正确
- 「删除」危险项红色文字 #C62828 + 红色图标 + 全宽分隔线隔离（A-06 决策 1）— 正确
- 各菜单项彩色图标区分合理（蓝/紫/绿/琥珀/红）

### ⚠️ 问题项（需修改）

3. **【轻微】BTOverflowMenu 图标位于右侧而非左侧**
   - **当前**: 每个菜单项为「文字（左）+ 彩色圆形图标（右）」
   - **期望**: A-06 确定的 BTOverflowMenu 布局为「彩色圆形图标（左）+ 文字（右）」— 图标在左侧引导视线
   - **说明**: 当前布局（文字左 + 图标右）在 iOS 中也有先例（如 UIMenu），不影响功能。但为了与 A-06 保持一致，建议调整为图标在左

4. **【轻微】Drill Card 2 缺少逐组明细行**
   - **当前**: 「中袋定位练习」仅显示累计进球，无 per-set 详情行
   - **期望**: 与 Drill Card 1 一样展示 3 组逐行明细（8/10, 9/10, 7/10）
   - **修复**: 为 Drill Card 2 添加 3 组详情行

5. **【轻微】缺少 Drill Card 3「走位控制练习」**
   - **当前**: 仅 2 个 Drill 卡片
   - **期望**: 提示词要求 3 个 Drill 卡片，第 3 个为「走位控制练习」4 组
   - **修复**: 添加第 3 个 Drill 卡片（不影响整体审核判定，数据完整性优化）

6. **【轻微】球台缩略图缺少橙色目标球**
   - **当前**: 缩略图中只有白色小圆点（仅母球）
   - **期望**: A-08 确立的斜线布局：母球白 #F5F5F5 (38%,70%) + 目标球橙 #F5A623 (58%,32%)
   - **修复**: 为缩略图中添加一个橙色小圆点代表目标球

### 💡 建议项（可选优化）

- 导航栏中间重复显示训练名「杆法专项训练」（nav title + header name），可考虑 nav 栏仅保留 ✕ 和「存为模版」，省略居中标题以减少冗余
- Overflow 菜单缺少背景遮罩（dimming overlay），添加半透明黑色遮罩可增强浮层焦点感
- 统计行最右侧「14:30·」被截断 — 属于横向滚动内容的预期行为，但可考虑减小 gap 或缩短数字让更多内容可见
