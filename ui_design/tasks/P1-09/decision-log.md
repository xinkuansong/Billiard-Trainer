# 设计决策日志 — P1-09

## 决策 1: 训练时长图表使用琥珀色柱体与成功率图表区分
- **上下文**: v1 中两个图表均使用绿色柱体，难以区分；设计文档要求训练时长使用琥珀色
- **决策**: 训练时长图表柱体 #F5A623 琥珀色，分类成功率图表柱体 #1A6B3C 品牌绿
- **理由**: 颜色语义区分 — 琥珀色 = 时间/时长数据，绿色 = 成功率/表现数据，与整体 Design Token 语义一致
- **日期**: 2026-04-04

## 决策 2: 环比指示器采用 emoji + 文字完整格式
- **上下文**: v1 仅显示百分比小标签，缺少 emoji 和「环比上周」上下文文字
- **决策**: 完整格式为「+X.X 小时 (+Y%) 👍 环比上周」，emoji 三色编码：👍 正向 / 😢 负向 / ⚖️ 持平
- **理由**: 与参考截图（训记 App）一致的信息密度，emoji 提供快速视觉扫描锚点
- **日期**: 2026-04-04

## 决策 3: 卡片左侧绿色边线装饰接受
- **上下文**: Stitch 为白色卡片添加了左侧品牌绿细竖线装饰，非设计文档原始要求
- **决策**: 接受此装饰效果
- **理由**: 统计页面内容区块密集，左侧绿色竖线有助于视觉引导和区块辨识，增强品牌感；此决策局限于统计页面
- **日期**: 2026-04-04

## 决策 4: Section 标题品牌绿色延续 P1-06 决策
- **上下文**: P1-06 确立数据统计类页面 Section 标题使用品牌绿 #1A6B3C
- **决策**: StatisticsView 的四个 Section 标题（训练概况/训练时长/分类成功率/各分类对比）全部使用品牌绿
- **理由**: 延续 P1-06 已确立的数据/统计类页面视觉模式，保持模块内一致性
- **日期**: 2026-04-04
