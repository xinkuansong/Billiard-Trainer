## ⚠️ 问题项（需修改）

### 问题 1（高优先级）：大标题 "动作库" 为绿色，应为黑色

- **期望**: iOS 原生大标题「动作库」34pt Bold Rounded **黑色** (#000000) 文字在 #F2F2F7 背景上（P0-01 决策 1）
- **当前**: 绿色文字（代码中 `text-[#005129]`）
- **对比**: P0-01 截图中「训练」为黑色大标题，P1-01 作为同级 Tab 根页面必须保持一致
- **影响范围**: 高 — 导航栏是跨页面一致性的基准

### 问题 2（中优先级）：Drill 卡片缺少 chevron 右箭头

- **期望**: 每张 BTDrillCard 右侧应有 chevron.right 箭头（`>`），作为可点击导航暗示（A-03 BTDrillCard 组件规范）
- **当前**: 卡片右侧只有 BTLevelBadge/PRO 标签，无 chevron
- **对比**: A-03 组件表截图中 BTDrillCard 明确展示了 `>` chevron
- **影响**: 缺少导航暗示会降低卡片的可发现性

### 问题 3（中优先级）：L0 BTLevelBadge 样式错误

- **期望**: L0「入门」应为 **实心绿色填充 #1A6B3C + 白色文字**（A-03 决策 2 — L0 使用实心填充强调"入门推荐"）
- **当前**: L0 使用浅绿底 + 绿色文字（代码 `bg-primary/10 text-primary`），与 L1 样式无法区分
- **对比**: A-03 组件表截图中 L0 入门 为深绿色实心 + 白字，与 L1 初学的浅绿底明显不同
- **影响**: L0 和 L1 视觉无区分，违反 A-03 确立的五级区分方案

## 💡 建议项（可选优化）

### 建议 1: Tab 栏样式与 P0-01 对齐

- 当前 Tab 栏使用毛玻璃效果 (`bg-white/80 backdrop-blur-xl`) + 大圆角顶部 (`rounded-t-3xl`) + 活跃项下方小圆点
- P0-01 的 Tab 栏为更简洁的白底样式，无毛玻璃/大圆角
- **建议**: 简化为与 P0-01 一致的标准 Tab 栏，减少视觉差异

### 建议 2: 卡片缩略图

- 当前卡片左侧带有台球实景照片缩略图（64pt 方形圆角），这不在 A-03 BTDrillCard 原始规范中
- **评估**: 照片缩略图丰富了视觉体验且增强品牌感，但需要确认后续一致性（P1-02 空状态、P1-03 详情页返回后列表等是否都带缩略图）
- **建议**: 可以保留作为 DrillListView 的增强，记录为设计决策

### 建议 3: "更多" 链接

- "准度训练" 分区标题右侧有绿色 "更多" 文字链接，设计文档未提及
- 功能上合理（分类内容较多时提供快捷入口），不影响整体设计
- 可以保留或去除，取决于产品需求
