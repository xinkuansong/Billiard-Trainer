# 设计决策日志 — P2-02

## 决策 1: 铅笔图标使用圆形容器样式
- **上下文**: 计划名称行的编辑图标样式——参考截图为内联铅笔图标（无背景），Stitch 生成为灰色圆形容器内的铅笔图标
- **决策**: 接受圆形容器样式
- **理由**: 圆形容器提供了更明确的可点击暗示，与 P0-03/P0-04 中的圆形图标按钮风格一致。参考截图来自不同 App，无需严格复刻
- **日期**: 2026-04-05

## 决策 2: 球台缩略图各 Drill 使用不同球体颜色
- **上下文**: 4 个 Drill 的缩略图球体颜色不同（红/橙/蓝/黄），是否统一为白+橙
- **决策**: 保留多色球体
- **理由**: 不同颜色增强了列表中各 Drill 的视觉区分度，用户无需阅读文字即可通过颜色快速定位。同时暗示了不同球种/训练类型。开发时可映射为球种颜色
- **日期**: 2026-04-05

## 决策 3: 底部工具栏「添加动作」使用品牌绿色
- **上下文**: 参考截图中底部中间按钮使用蓝色，我们的设计使用品牌绿
- **决策**: 使用品牌绿 #1A6B3C
- **理由**: 与全 App 品牌色体系一致。P0-03/P0-04 底部工具栏中间按钮使用蓝色是因为沿用了系统色（添加按钮），但此处为计划编辑场景，品牌绿更合适
- **日期**: 2026-04-05

## 决策 4: DESIGN.md 偏离延续既有模式
- **上下文**: Stitch DESIGN.md 延续 "The Tactical Gentleman" / "High-End Editorial" 主题，No-Line Rule、Glass & Gradient Rule 等偏离
- **决策**: 延续 A-01 决策 3 — 以 screen.png 渲染效果为准，DESIGN.md 仅作参考
- **理由**: 渲染层面品牌色、卡片风格、导航模式均正确
- **日期**: 2026-04-05
