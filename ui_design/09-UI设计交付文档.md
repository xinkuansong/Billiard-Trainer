## 1. 产品概述

**一句话定位**：面向中式八球和九球爱好者的台球训练记录与角度感知学习工具。

**目标用户**：
- 台球爱好者（入门到高级），日常在球馆练球
- 主打中式八球和九球两大球种

**核心价值主张**：
- **动作库**：72 个台球训练项目（Drill），含球台动画图解、分级教学
- **训练记录**：像健身 App 记录动作一样记录每次练球（组数 × 每组球数）
- **角度训练**：独创角度感知测试 + 进球点对照表，帮助建立角度直觉
- **数据复盘**：日历回顾 + 成功率趋势统计

**概念映射**（参照健身训练 App「训记」）：

| 健身 App 概念 | 球迹对应 |
|---|---|
| 健身动作（如杠铃卧推） | 台球训练项目 Drill（如五分点定杆练习） |
| 动作分类（胸/背/腿） | 训练分类（准度/杆法/走位，共 8 大类） |
| 训练计划（胸肌日） | 台球训练计划（今日专项走位训练） |
| 组数 × 重复次数 | 组数 × 每组球数（如 3 组 × 15 球） |
| 进度（力量增长） | 进度（完成率/成功率提升） |

**技术平台**：iOS 17+，SwiftUI 原生，支持 Light/Dark Mode。

## 1.5 参考 App 设计语言摘要

球迹的 UI 布局以健身训练 App「训记」为主要参考对象。以下提炼训记的核心设计模式，作为球迹各页面的布局基准。参考截图存放于 `ref-screenshots/` 目录。

### 导航模式

| 层级 | 模式 | 球迹对应 |
|---|---|---|
| 一级 | 5 Tab 底部导航（训练/挑战/动作/历史/我的） | 球迹 5 Tab（训练/动作库/角度/历史/我的） |
| 二级 | 页内水平文本标签（官方计划/个人模版/个人计划AI），活跃项带下划线 | BTSegmentedTab 组件 |
| 三级 | 左侧边栏分类导航（动作库按部位分类） | DrillListView 的分类 Chip 行（球迹 Drill 数量较少，不采用侧边栏） |
| 详情 | 全屏 Sheet/Push + 返回箭头或「返回」胶囊按钮 | 标准 NavigationStack push |
| 模态 | 居中白色卡片 + 暗色模糊背景（确认弹窗） | 系统 Alert + 自定义 Sheet |
| 上下文 | 三点「⋮」溢出菜单（浮层卡片+图标行列表） | BTOverflowMenu 组件 |
| 全局 | 浮动训练指示器（训练进行中跨 Tab 显示） | BTFloatingIndicator 组件 |

## 2. 全局导航与信息架构

### 2.1 底部 Tab 导航

App 使用 5 Tab 底部导航栏，每个 Tab 内使用独立的 NavigationStack（导航深度一般 2-3 层）。

| Tab 序号 | 名称 | SF Symbol 图标 | 使用频率 | 定位 |
|---|---|---|---|---|
| 1 | 训练 | `dumbbell.fill` | 每次练球必用 | 日常最高频入口 |
| 2 | 动作库 | `list.bullet.rectangle` | 中频 | 查资料、选 Drill |
| 3 | 角度 | `angle` | 中频 | 专项角度感知训练 |
| 4 | 历史 | `clock.arrow.circlepath` | 低频 | 复盘与统计 |
| 5 | 我的 | `person.circle` | 低频 | 个人信息与设置 |

### 2.2 首次启动流程

```
App 启动
  → 判断是否完成引导
    → 未引导：显示 Onboarding（3 页引导）→ 可选登录 → MainTabView
    → 已引导：直接进入 MainTabView（默认训练 Tab）
```

#### 3.1.1 TrainingHomeView — 今日训练首页

**文件路径**：`QiuJi/Features/Training/Views/TrainingHomeView.swift`

**功能描述**：训练 Tab 的根页面，承载今日训练安排、即将到来的训练预告和计划浏览三大信息区块，引导用户快速开始训练。

> **布局参考**：`ref-screenshots/01-training-home/01-home-with-plan.png`（有计划态）、`ref-screenshots/01-training-home/04-home-scrolled-plans.png`（计划浏览区）

**页面元素**（有激活计划时，从上到下）：

- **导航栏**：大标题「训练」（34pt Bold Rounded），右上角两个图标按钮——「朋友」（社交入口）+「⋯」溢出菜单（设置、管理计划）
- **今日安排区**（Section Header：「今日安排」16pt Bold）：
  - 每项为白色卡片（圆角 16pt，内边距 16pt）：
    - 左侧：训练名称（20pt Bold，如「五分点专项#1」）
    - 副标题：计划来源 + 组数信息（13pt 辅助色，如「新手入门计划 · 3 练」）+ Pro 胶囊标签（如适用）
    - 右侧：操作入口——「GO!」动作按钮 或 三点「⋮」溢出菜单
  - 支持多张卡片（今日有多项训练时垂直堆叠）
- **即将到来区**（Section Header：「即将到来」16pt Bold）：
  - 同样卡片样式，预览下一个训练日的安排（灰色文字标注日期）
  - 若无后续安排则不显示此分区
- **计划浏览区**：
  - **BTSegmentedTab 水平标签**：「官方计划 / 自定义模版」，活跃项带品牌色下划线
  - **筛选 Chip 行**（横向滚动）：全部 / 入门 / 初级 / 中级 / 高级 / 综合
    - 选中 Chip：黑色/深色填充 + 白字；未选中：灰色边框
  - **计划卡片列表**：全宽卡片，每张含：
    - 左侧计划封面缩略图（80pt 方形圆角）
    - 右侧：计划名称（16pt Bold）+ 描述副标题（13pt 辅助灰色）
    - Pro 锁定标识（如适用）
- **底部主按钮**（固定底部）：「开始训练」品牌绿色填充全宽按钮，50pt 高，16pt 距底部安全区

**无激活计划时（空状态）**：
- BTEmptyState 组件：SF Symbol 图标 + 「还没有训练计划」标题 + 引导副标题
- 两个按钮：「选择训练计划」（primary）/「自由记录」（secondary）

**交互说明**：
- 点击今日安排卡片的「GO!」→ `fullScreenCover` 弹出 ActiveTrainingView，自动加载该训练的 Drill 列表
- 点击计划卡片 → push 到 PlanDetailView
- 点击 BTSegmentedTab 切换 → 更新下方计划列表
- 下拉可刷新
- 训练中时，页面底部显示 BTFloatingIndicator

**空状态**：无激活计划时显示引导选择计划或自由记录
**Freemium 限制**：无

#### 3.1.2 PlanListView — 训练计划列表

**文件路径**：`QiuJi/Features/Training/Views/PlanListView.swift`

**功能描述**：展示所有可用的训练计划，分为官方计划和自定义计划。

**页面元素**：
- **导航栏**：标题「训练计划」，右上角「新建」按钮（创建自定义计划）
- **官方计划区**：
  - 按等级分组（入门 / 初级 / 中级 / 高级 / 综合）
  - 每套计划卡片：名称、等级标签、周期说明、Pro 锁标识（付费计划）
  - 共 6 套计划
- **自定义计划区**：
  - 用户创建的计划列表
  - 支持滑动删除（带确认 Alert）

**交互说明**：
- 点击官方计划 → push 到 PlanDetailView
- 点击「新建」→ push 到 CustomPlanBuilderView
- 长按/滑动自定义计划 → 删除确认弹窗

**空状态**：自定义计划区为空时可不显示该分区
**Freemium 限制**：
- 免费计划：新手入门 8 周计划、基础杆法专项（2 套）
- Pro 计划（显示锁标识）：走位突破、中级综合突破、高级加塞与多库、全能综合计划（4 套）

#### 3.1.3 PlanDetailView — 官方计划详情

**文件路径**：`QiuJi/Features/Training/Views/PlanDetailView.swift`

**功能描述**：展示单个官方训练计划的完整内容和进度。

**页面元素**：

**交互说明**：
- 点击「开始此计划」→ Alert 确认 → 激活计划 → 返回 TrainingHomeView

#### 3.1.4 CustomPlanBuilderView — 自定义计划创建

**文件路径**：`QiuJi/Features/Training/Views/CustomPlanBuilderView.swift`

**功能描述**：创建或编辑用户自定义训练计划。

**页面元素**：

#### 3.1.5 ActiveTrainingView — 全屏训练记录流 ★

**文件路径**：`QiuJi/Features/Training/Views/ActiveTrainingView.swift`

**功能描述**：训练进行中的核心记录界面，全屏模式。采用**双层结构**：训练总览视图展示所有 Drill 列表和整体进度，单项记录视图展开单个 Drill 的逐组输入。这是产品最重要的交互页面。

> **布局参考**：`ref-screenshots/02-training-active/03-active-main.png`（总览）、`04-active-set-progress.png`（单项记录）

##### A. 训练总览视图

**页面元素**（从上到下）：
- **顶部计时栏**：
  - 左侧：大号训练计时器，品牌绿色，28pt 等宽字体（如 `00:12:34`）
  - 右侧：操作图标行——闹钟（组间休息设置）/ 历史 / 排序 / 保存，32pt 圆形图标按钮
  - 第二行左侧：训练名称（可编辑，带铅笔图标），右侧：进度统计 `X/Y 组  X/Y 项目`
- **Drill 列表**（ScrollView，垂直排列）：
  - 每项为 BTExerciseRow 白色卡片（80pt 高）：

#### 3.3.2 AngleTestView — 角度测试答题 ★

**文件路径**：`QiuJi/Features/AngleTraining/Views/AngleTestView.swift`

**功能描述**：核心角度感知训练功能。随机生成球台布局，用户目测切入角并输入估计值。

**页面元素**：
- **顶部进度条**：当前题号 / 总题数
- **球台 Canvas 区域**（BTAngleTestTable）：
  - 俯视标准球台（6 个袋口可见）
  - 目标球 + 袋口方向标记箭头
  - 母球位置（每题随机生成）
  - 切入角范围：5°-85°，以 5° 为步进
- **输入区域**：
  - TextField 数字键盘输入估计角度（0-90°整数）
  - 主按钮「确认」（BTButtonStyle.primary）
- **结果反馈（确认后动画展示）**：
  - 正确击球线高亮动画（从母球到目标球）
  - 目标球面接触点闪烁标注
  - 可选幽灵球位置显示
  - 误差文案：「你答了 35°，实际是 30°，误差 5°」
  - 判定：正确（误差 ≤ ±3°）/ 不正确
  - 结果气泡 + 颜色反馈
- **下一题按钮**
- **每轮结束总结**：正确率、平均误差

**自适应出题逻辑**（影响 UI 展示）：
- 出题权重 = 基础权重 30% + 误差权重 70%
- 误差大的角度区间出题概率更高
- 角袋约 60% / 中袋约 40%
- 推荐每轮 20 题（约 5-8 分钟）
- 判定容差：±3°

**交互流程**：
```
进入角度测试
  → 显示球台：目标球 + 袋口标记 + 母球
```

## 7. 设计约束与建议

### 7.1 使用场景约束

- **练球场景**：用户在球馆使用 App，环境光线可能不足
- **单手操作为主**：一只手拿手机记录，另一只手拿球杆
- **时间碎片化**：每组球间隙记录，需要极快的操作速度

### 7.2 交互设计原则

1. **3 步到达核心操作**：打开 App → 今日卡 → 开始训练 → 第一个 Drill 记录界面
2. **按钮最小 44pt**：所有可点击目标满足 HIG 最低触控尺寸
3. **底部操作优先**：主操作按钮置于屏幕下方拇指热区
4. **减少文字输入**：训练记录中不强制输入文字，进球数用大按钮快速点击/加减
5. **离线优先**：所有核心功能（动作库、训练记录、角度训练）可完全离线使用

### 7.3 视觉设计要求

1. **Light / Dark Mode 双模式**：所有页面必须适配
2. **HIG 一致性**：遵循 Apple Human Interface Guidelines
   - 使用 SF Symbols 图标体系
   - 使用系统字体（SF Pro / SF Pro Rounded）
   - 标准 Sheet / NavigationStack / Alert 交互模式
3. **品牌色**：当前主色为绿色（`#1A6B3C` / `#25A25A`），与台球桌绿色台面呼应
4. **球台动画规范**（待设计出稿）：
   - 俯视视角
   - 母球白色、目标球橙色、台面绿色、库边棕色
   - 路径动画多阶段时序：击球前 → 接触 → 母球走位 → 目标球走位
   - 接触点视觉强调（闪烁或缩放）
5. **角度测试结果动画**：
   - 正确击球线渐显高亮
   - 目标球接触点闪烁标注
   - 幽灵球位置渐显（可选功能）

## 附录

### A. 核心用户路径

**路径一：按计划练球（每次练球必用）**
```
打开 App → 今日安排卡片「GO!」→ 全屏训练总览
  → 点击 Drill → 单项记录视图（组数网格输入）
    → 逐组输入进球数（44pt 输入框）→ 勾选完成
      → 组间休息倒计时（可选）→ 下一组/下一项
        → 全部完成 → 心得（可跳过）→ 训练总结
          → 生成分享图（可选）→ 保存
```

**路径二：查技术动作**
```
动作库 Tab → 选分类 → 选子类 → Drill 详情
  → 查看动画 + 教学 → 加入训练计划
```

**路径三：角度训练**
```
角度 Tab → 「角度测试」→ 看球台图 → 输入角度 → 确认
  → 结果动画（击球线 + 接触点）→ 下一题 → 循环
```

**路径四：复盘统计**
```
历史 Tab → 日历选日期 → 训练详情 → 统计图表
  → 发现薄弱项 → 跳转动作库 → 加入计划
```
