# 产品上下文（持续更新）

> 本文件在讨论中随共识演进更新，供智能体快速加载「当前产品共识」。

## 工作名称

台球训练 / 教学类 iOS App（与仓库名 `billiard_trainer` 一致，正式产品名待定）。

## 当前产品定位（2026-03-23 最新）

**定位**：面向中国大陆市场的台球训练伴侣 App，以「台球动作库」为内核，整合系统化训练内容、训练计划、训练记录与数据分析。

**产品核心参照**：训记（健身记录 App）—— 台球 Drill 视同健身动作，建立完整内容库

**球种范围**：中式台球 + 9球（v1.0），斯诺克 defer

**信息架构**：5 Tab（训练 / 动作库 / 角度 / 历史 / 我的）

**角度感知训练**（第5个Tab，产品独创）：角度测试（自适应权重，专业容差±3°）+ 进球点对照表（sin(α)×R，纯本地计算）

**动作库规模**：约72项Drill，8大类（基础功/准度/杆法/分离角/走位/控力/特殊球路/综合球形）

**技术栈**：SwiftUI + iOS 17 + SwiftData（本地离线）+ LeanCloud（用户认证 + 私有数据同步）+ CloudKit 公开库（Drill 内容热更新）

**目标用户**：C端为主——普通爱好者 + 进阶/职业玩家（培训机构用C端功能，v2.0再做B端）

**开发方式**：个人开发，Claude Code + Cursor，无硬性时间限制

## 已确认决策

- 产品定位：训练工具，非娱乐游戏
- 不做在线对战/娱乐、通用社交广场、直播、跳球
- v1.0 C端，B端 defer to v2.0
- 球种：中式台球 + 9球（v1.0），斯诺克 defer
- 登录方式：微信登录 + 手机验证码 + Sign in with Apple（三种）
- 用户数据：LeanCloud（支持微信/短信/Apple三种登录）
- 内容数据：CloudKit 公开库（Drill 内容可热更新，无需发版）
- 离线优先：无网络（球馆）下训练记录、动作库、角度训练均可完整使用
- 教学视频纳入产品，v1.0位置预留，视频来源由用户解决
- Drill 详情页含球台路径动画（SwiftUI Canvas）
- 训练记录模型：球种 + Drill列表 × 组数 × 每组进球数（数字键盘）+ 心得
- 官方训练计划：6套，基于多源调研（新锐台球学院+伟林+王桁+BU+BIZU+B&PC）
